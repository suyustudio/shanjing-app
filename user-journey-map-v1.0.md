## 1. 旅程地图概述

### 1.1 什么是用户旅程地图

用户旅程地图（User Journey Map）是一种可视化工具，用于描述用户在使用产品过程中各个阶段的体验、情感、痛点和机会点。

### 1.2 山径APP旅程地图设计原则

| 原则 | 说明 |
|------|------|
| **场景驱动** | 基于真实徒步场景设计旅程 |
| **情感导向** | 关注用户情绪变化，识别低谷和高峰 |
| **全链路覆盖** | 从发现到分享的完整闭环 |
| **痛点聚焦** | 识别关键痛点，指导产品优化 |

### 1.3 目标用户画像回顾

**核心用户**: 25-32岁城市女性白领
- **特征**: 手机依赖度高、安全意识强、审美要求高
- **场景**: 周末1天往返，5-10km轻度徒步，1-3人小团体
- **痛点排序**: 怕迷路 > 找不到靠谱路线 > 担心安全 > 分享麻烦
