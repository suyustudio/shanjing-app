## 1. 设计原则

### 1.1 品牌调性

基于PRD定义的产品定位，山径APP的设计语言应传达以下品牌特质：

| 维度 | 关键词 | 设计表达 |
|------|--------|----------|
| **自然** | 山野、清新、生机 | 源自自然的色彩，有机的圆角，呼吸感的留白 |
| **安心** | 可靠、安全、专业 | 清晰的信息层级，稳定的布局，明确的反馈 |
| **诗意** | 雅致、文艺、温度 | 精致的排版，细腻的动效，有温度的文案 |
| **年轻** | 活力、简洁、现代 | 现代的界面语言，克制的装饰，高效的操作 |

### 1.2 设计目标

1. **降低认知负荷** - 户外场景下，用户需要快速获取关键信息
2. **建立安全感** - 通过稳定、可靠的视觉语言传递"不迷路"的承诺
3. **激发探索欲** - 用自然美感的视觉呈现激发用户对户外的向往
4. **保证可读性** - 户外强光环境下，信息依然清晰可读

## 2. 色彩系统

### 2.1 色彩理念

山径的色彩系统源于自然山野的色谱：
- **主色调** - 取自山林的苍翠与晨雾的柔和
- **辅助色** - 取自天空、土壤、岩石的自然色彩
- **功能色** - 在保持自然感的同时确保功能性

### 2.6 深色模式适配

#### 深色模式色彩映射

深色模式下，色彩系统保持色相不变，调整明度：

| 浅色模式 | 深色模式 | 用途 |
|----------|----------|------|
| `#FFFFFF` (White) | `#0F1419` (Dark BG) | 页面背景 |
| `#F9FAFB` (Gray-50) | `#1A1F24` (Dark Card) | 卡片背景 |
| `#F3F4F6` (Gray-100) | `#252B32` (Dark Elevated) | 悬浮卡片 |
| `#111827` (Gray-900) | `#FFFFFF` | 主文字 |
| `#374151` (Gray-700) | `#E5E7EB` (Gray-200) | 标题文字 |
| `#4B5563` (Gray-600) | `#9CA3AF` (Gray-400) | 正文文字 |
| `#6B7280` (Gray-500) | `#6B7280` | 次要文字 |
| `#9CA3AF` (Gray-400) | `#4B5563` (Gray-600) | 占位符 |
| `#2D968A` (Primary-500) | `#4DB6AC` (Primary-400) | 主按钮、强调 |
| `#1D756B` (Primary-700) | `#80CBC4` (Primary-300) | 链接文字 |

#### 深色模式专用色

| 名称 | 色值 | 用途 |
|------|------|------|
| **Dark BG** | `#0F1419` | 主背景色 |
| **Dark Card** | `#1A1F24` | 卡片背景 |
| **Dark Elevated** | `#252B32` | 悬浮/选中背景 |
| **Dark Border** | `#2D3748` | 边框、分割线 |

#### 深色模式功能色调整

| 功能 | 浅色模式 | 深色模式 | 调整说明 |
|------|----------|----------|----------|
| 成功 | `#4CAF50` | `#66BB6A` | 提高明度，保持可见 |
| 警告 | `#FFC107` | `#FFD54F` | 提高明度 |
| 错误 | `#EF5350` | `#EF5350` | 保持不变，已足够醒目 |
| 信息 | `#3B9EFF` | `#64B5F6` | 提高明度 |

#### 深色模式使用原则

1. **保持对比度** - 文字与背景对比度至少 4.5:1
2. **降低饱和度** - 大面积色彩在深色模式下降低饱和度，减少视觉疲劳
3. **增加层次感** - 使用 elevation 色阶区分层级，而非阴影
4. **保持品牌识别** - 主色调色相不变，仅调整明度

## 9. 焦点状态设计 (Week 3 Day 3)

遵循 WCAG 2.2 无障碍标准，确保键盘导航可见性。

### 9.1 按钮焦点状态

| 属性 | 值 | 说明 |
|------|-----|------|
| **outline** | `2px solid` | 轮廓线宽度 |
| **outline-color** | `--color-primary-500` | 与主色一致 |
| **outline-offset** | `2px` | 轮廓与按钮间距 |
| **border-radius** | 继承按钮圆角 | 保持视觉一致 |

```css
button:focus-visible {
  outline: 2px solid var(--color-primary-500);
  outline-offset: 2px;
}
```

### 9.2 输入框焦点状态

| 属性 | 值 | 说明 |
|------|-----|------|
| **border** | `2px solid` | 边框加粗 |
| **border-color** | `--color-primary-500` | 主色高亮 |
| **box-shadow** | `0 0 0 3px rgba(45, 150, 138, 0.15)` | 柔和光晕 |

```css
input:focus {
  border: 2px solid var(--color-primary-500);
  box-shadow: 0 0 0 3px rgba(45, 150, 138, 0.15);
}
```

### 9.3 链接焦点状态

| 属性 | 值 | 说明 |
|------|-----|------|
| **text-decoration** | `underline` | 下划线 |
| **text-decoration-thickness** | `2px` | 下划线粗细 |
| **outline** | `none` | 链接使用下划线替代轮廓 |

```css
a:focus-visible {
  text-decoration: underline;
  text-decoration-thickness: 2px;
  outline: none;
}
```

### 9.4 复选框/单选框/开关焦点状态

| 组件 | 焦点样式 |
|------|----------|
| **复选框** | `outline: 2px solid var(--color-primary-500); outline-offset: 2px;` |
| **单选框** | `outline: 2px solid var(--color-primary-500); outline-offset: 2px;` |
| **开关** | `outline: 2px solid var(--color-primary-500); outline-offset: 2px;` |

### 9.5 WCAG 2.2 合规说明

| 要求 | 实现方式 |
|------|----------|
| 可见性 | 焦点指示器与背景对比度 ≥ 3:1 |
| 尺寸 | 轮廓线厚度 ≥ 2px，覆盖元素周长 |
| 不遮挡 | outline-offset 确保不覆盖内容 |
| 键盘触发 | 使用 `:focus-visible` 仅键盘显示 |

### 9.3 语义色映射 (暗黑模式)

| 语义 | 浅色模式 | 暗黑模式 |
|------|----------|----------|
| **成功** | `#4CAF50` | `#3FB950` |
| **警告** | `#FFC107` | `#D29922` |
| **错误** | `#EF5350` | `#F85149` |
| **信息** | `#3B9EFF` | `#58A6FF` |

### 9.4 暗黑模式设计原则

1. **降低亮度**: 主背景 `#0F1419` 接近纯黑但不刺眼
2. **提升饱和度**: 品牌色保持色相，适当提升饱和度保证可见
3. **层级清晰**: 使用 `#161B22` → `#1C2128` → `#21262D` 区分层级
4. **对比度保证**: 所有文字对比度 ≥ 4.5:1

### 9.5 地图组件暗黑模式

#### 高德地图暗黑模式 API 调用

```javascript
// 初始化时设置地图样式为暗黑模式
const map = new AMap.Map('container', {
  mapStyle: 'amap://styles/dark',  // 官方暗黑主题
  // 或自定义样式
  mapStyle: 'amap://styles/xxx'    // 自定义样式 ID
});

// 动态切换
map.setMapStyle('amap://styles/dark');
```

#### 地图样式切换方案

| 场景 | 处理方式 |
|------|----------|
| **跟随系统** | 监听 `prefers-color-scheme`，自动切换 `dark`/`normal` |
| **手动切换** | 设置页提供开关，切换时调用 `setMapStyle()` |
| **过渡动画** | 切换时添加 300ms 淡入淡出，避免闪烁 |
