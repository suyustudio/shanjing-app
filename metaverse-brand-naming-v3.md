# 「元宇宙UGC平台」品牌命名方案 - 第三版

## 参考对象分析

### TikTok
- 无意义，拟声词
- 2音节，节奏感强（TiK-Tok）
- 全球发音一致
- 年轻、活泼、快速

### Discord
- 原意"不和"，但游戏社区很酷
- 2音节，重音在后
- 有画面感（声音不和谐）
- 反差感

### Notion
- 原意"概念"，但笔记工具很妙
- 2音节，柔和
- 抽象但贴切
- 知识工作者共鸣

### 共同点
| 特征 | 说明 |
|------|------|
| **2音节** | 最短的记忆单元 |
| **无特定文化标签** | 不是日本、不是欧洲、不是美国 |
| **有节奏感** | 像心跳、像鼓点 |
| **无意义但顺口** | 不解释也能念 |
| **可当动词** | "TikTok一下"、"Discord我" |
