## 二、详细 Review 结果

### 2.1 设计一致性审查

#### 2.1.1 问题清单

| 序号 | 问题描述 | 严重程度 | 涉及文档 | 具体位置 |
|------|----------|----------|----------|----------|
| DC-01 | 难度等级色值不统一 | 🔴 高 | design-system-v1.0.md vs design-hifi-v1.0.md | 设计系统定义 `#8BC34A` 为轻度难度色，但高保真设计使用 `#FFC107`（秋叶黄）作为轻度 |
| DC-02 | 字体使用不一致 | 🟡 中 | prototype-lofi-v1.0.md | 低保真原型使用 `sp/dp` 单位，但设计系统使用 `px` |
| DC-03 | Tab栏高度定义冲突 | 🟡 中 | design-hifi-v1.0.md | 路线详情页Tab栏高度44px，但我的页面未定义Tab栏高度 |
| DC-04 | 卡片圆角不统一 | 🟡 中 | design-hifi-v1.0.md | 路线卡片使用12px圆角，搜索结果卡片使用8px圆角 |
| DC-05 | 按钮高度不一致 | 🟡 中 | design-hifi-v1.0.md | 主要按钮48px，但登录按钮52px（引导页）vs 48px（登录页） |

#### 2.1.2 改进建议

**DC-01 难度等级色值统一**

```
当前问题:
- 设计系统: 轻度 = #8BC34A (黄绿)
- 高保真设计: 轻度 = #FFC107 (秋叶黄/琥珀)

建议统一为:
- 休闲: #4CAF50 (翠竹绿500)
- 轻度: #8BC34A (黄绿) - 保持设计系统定义
- 进阶: #FF9800 (橙黄)
- 挑战: #F44336 (红色)

原因: #8BC34A 与 #4CAF50 形成更好的绿色渐变层次，符合自然调性
```

**DC-04 卡片圆角标准化**

建议建立卡片圆角规范：
| 卡片类型 | 圆角值 | Token |
|----------|--------|-------|
| 大卡片（路线卡片） | 12px | --radius-lg |
| 中卡片（功能卡片） | 12px | --radius-lg |
| 小卡片（搜索结果） | 8px | --radius-md |
| 列表项 | 8px | --radius-md |

#### 2.2.2 改进建议

**UX-01 偏航处理流程优化**

当前设计仅描述了偏航检测和提醒，缺少完整的重新规划交互：

```
建议增加以下流程:
1. 偏航检测 (>50m)
2. 震动+语音提醒: "您已偏离路线"
3. 弹窗提示:
   ┌─────────────────────────────────┐
   │  ⚠️ 偏离路线                    │
   │                                 │
   │  您已偏离原路线约60米           │
   │                                 │
   │  [返回原路线]  [重新规划]       │
   │                                 │
   │  [继续当前方向]                 │
   └─────────────────────────────────┘
4. 选择"重新规划"后，计算新路线并更新导航
```

**UX-06 SOS按钮防误触设计**

当前设计仅提到"长按触发: 800ms"，建议增加完整流程：

```
建议设计:
1. 按钮设计: 红色圆形，64px，"SOS"文字
2. 长按触发: 800ms（已定义）
3. 长按过程中:
   - 按钮逐渐缩小 (scale 1 → 0.9)
   - 环形进度条显示长按进度
   - 达到阈值后震动反馈
4. 触发后二次确认弹窗:
   ┌─────────────────────────────────┐
   │  ⚠️ 紧急求助                    │
   │                                 │
   │  将向您的紧急联系人发送         │
   │  当前位置信息                   │
   │                                 │
   │  [取消]        [确认发送]       │
   └─────────────────────────────────┘
```

**UX-02 离线包下载进度设计**

为下载按钮增加进度状态：

下载状态设计:
