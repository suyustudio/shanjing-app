# Flutter Design Tokens

## 色彩系统

### 品牌色
```dart
class BrandColors {
  static const primary = Color(0xFF2D968A);      // 山青色 - 设计系统 v1.0 主色
  static const primaryLight = Color(0xFF4DB6AC); // 山青色浅色
  static const primaryDark = Color(0xFF1F6B62);  // 山青色深色
}
```

### 背景色
```dart
class BackgroundColors {
  static const base = Color(0xFFFFFFFF);
  static const elevated = Color(0xFFF5F5F5);
  static const pressed = Color(0xFFE5E5E5);
}
```

### 文字色
```dart
class TextColors {
  static const primary = Color(0xFF262626);
  static const secondary = Color(0xFF8C8C8C);
  static const tertiary = Color(0xFFBFBBB);
  static const inverse = Color(0xFFFFFFFF);
}
```
