## 1. 设计概述

### 1.1 功能定位

轨迹采集功能是山径APP的核心内容生产工具，面向**认证领队**和**运营人员**，用于采集高质量路线数据。该功能需要：

- 高精度GPS轨迹记录（每秒1点）
- 丰富的POI标记能力（8种类型）
- 照片与POI自动关联
- 离线数据缓存与断点续传

### 1.2 用户画像

| 角色 | 使用场景 | 核心诉求 |
|------|----------|----------|
| **认证领队** | 带队前探路，完善路线信息 | 快速标记、省电、数据准确 |
| **运营人员** | 上线新路线前的专业采集 | 完整数据、批量上传、审核流程 |
| **资深用户** | 分享优质私藏路线 | 简单易用、有成就感 |

### 1.3 设计目标

1. **单手操作友好** - 户外场景下单手操作，按钮集中在底部
2. **户外可见性** - 强光下依然清晰可读，高对比度设计
3. **省电优先** - 深色模式、减少动画、降低亮度
4. **防误触设计** - 关键操作（结束录制）需要二次确认
5. **快速标记** - POI标记流程不超过3秒

### 1.4 使用场景

```
【户外采集场景特点】
- 环境光线：强烈阳光、树荫斑驳、逆光
- 操作条件：戴手套、单手操作、可能手持登山杖
- 网络状态：弱网或无网，需离线工作
- 电量焦虑：担心手机没电，需要省电模式
- 注意力：需同时关注路况，不能长时间看手机
```

## 2. 设计原则

### 2.1 单手操作设计

```
┌─────────────────────────────┐
│                             │
│       地图显示区域           │  ← 上部：信息展示
│       (可点击查看详情)        │
│                             │
├─────────────────────────────┤
│  [数据面板 - 距离/时长/海拔]  │  ← 中部：关键数据
├─────────────────────────────┤
│  [标记] [拍照] [暂停] [结束] │  ← 底部：操作按钮
│        底部操作栏            │     (拇指轻松触及)
└─────────────────────────────┘
```

**设计要点：**
- 主要操作按钮位于底部 1/3 区域
- 按钮尺寸 ≥ 56px，间隔 ≥ 12px
- 悬浮操作按钮（FAB）直径 72px
- 避免顶部导航栏的频繁操作

### 2.2 户外可见性设计

| 设计维度 | 浅色模式 | 深色模式（推荐） |
|----------|----------|------------------|
| **背景** | 纯白 `#FFFFFF` | 深灰 `#0F1419` |
| **主文字** | 深灰 `#111827` | 纯白 `#FFFFFF` |
| **数据数字** | 品牌色 `#2D968A` | 品牌色 `#4DB6AC` |
| **地图轨迹** | 品牌色 `#2D968A` | 高亮青 `#4DB6AC` |
| **POI标记** | 多彩高饱和 | 高亮发光效果 |
| **对比度** | ≥ 4.5:1 | ≥ 7:1（夜间推荐） |

### 2.3 省电设计策略

| 策略 | 实现方式 |
|------|----------|
| **深色模式优先** | 默认使用深色主题，OLED屏幕节电30%+ |
| **降低刷新率** | 录制中界面锁定30fps，地图按需渲染 |
| **传感器优化** | GPS 1秒间隔，其他传感器按需唤醒 |
| **屏幕常亮** | 录制中保持常亮，但亮度自动降低至40% |
| **自动休眠** | 暂停10分钟后自动结束录制 |
| **电池警告** | 电量<20%时提醒保存，<10%强制结束 |

## 6. 动画与交互

### 6.1 录制状态指示

**录制中动画：**
- 状态栏圆点：呼吸灯效果（1.5s循环）
- 轨迹线：实时绘制，有流动效果
- 当前位置：脉冲波纹效果

**时间规格：**
```css
/* 录制指示器呼吸动画 */
@keyframes recording-pulse {
  0% { opacity: 1; transform: scale(1); }
  50% { opacity: 0.5; transform: scale(0.9); }
  100% { opacity: 1; transform: scale(1); }
}
/* 时长：1.5s，无限循环 */

/* 当前位置脉冲 */
@keyframes location-pulse {
  0% { transform: scale(1); opacity: 1; }
  100% { transform: scale(2); opacity: 0; }
}
/* 时长：2s，无限循环 */
```
