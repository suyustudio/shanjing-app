## 4. 导出规格

### 4.1 iOS 规格

| 倍率 | 后缀 | 说明 |
|------|------|------|
| 1x | @1x | iPhone SE/老设备 |
| 2x | @2x | iPhone 8/Plus/X/11/12/13/14/15 |
| 3x | @3x | iPhone Plus/Pro Max |

**示例命名:**
- ic_comment@1x.png
- ic_comment@2x.png
- ic_comment@3x.png
- ic_comment.svg (矢量源文件)

### 4.2 Android 规格

| 密度 | 倍率 | 文件夹 |
|------|------|--------|
| mdpi | 1x | drawable-mdpi/ |
| hdpi | 1.5x | drawable-hdpi/ |
| xhdpi | 2x | drawable-xhdpi/ |
| xxhdpi | 3x | drawable-xxhdpi/ |
| xxxhdpi | 4x | drawable-xxxhdpi/ |

**示例命名:**
- ic_comment.png (各密度)
- ic_comment.xml (矢量)

### 4.3 Web 规格

| 格式 | 用途 | 备注 |
|------|------|------|
| SVG | 图标 | 可缩放 |
| PNG | 照片/复杂图形 | 透明背景 |
| WebP | 优化加载 | 现代浏览器 |
| Lottie JSON | 动画 | 复杂动画 |

## 6. 资源命名规范

### 6.1 图标命名

格式: `[类型]_[名称]_[状态]_[尺寸]`

示例:
- ic_send_24px
- ic_like_filled_24px
- ic_star_empty_24px
- btn_follow_default
- btn_following_hover

### 6.2 插画命名

格式: `[场景]_[类型]_[尺寸]`

示例:
- empty_favorites_120px
- guide_photo_upload_200px
- banner_social_feature_375px

### 6.3 照片占位

格式: `placeholder_[类型]_[尺寸]`

示例:
- placeholder_photo_grid_120px
- placeholder_avatar_80px
- placeholder_cover_375px
