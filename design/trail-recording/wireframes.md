## 尺寸速查表

| 元素 | 尺寸 | 备注 |
|------|------|------|
| 状态栏 | 44px | 固定高度 |
| 底部操作栏 | 122px | 72px按钮 + 50px安全区 |
| 数据面板 | 80px | 固定高度 |
| 地图区域 | 剩余空间 | 自适应 |
| 圆形按钮 | 72×72px | 底部操作按钮 |
| 方形按钮 | 48px高 | 标准按钮 |
| 卡片 | 100px高 | 采集记录卡片 |
| 图标背景 | 56×56px | POI类型选择 |
| 图标 | 48×48px | POI图标 |
| 页面边距 | 16px | 左右边距 |
| 按钮间距 | 16px | 底部按钮间距 |
