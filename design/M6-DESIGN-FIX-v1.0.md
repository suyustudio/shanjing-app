#### 1.2 点赞动画效果

**动画时序:**
```
0ms     → 点击触发
0-50ms  → scale(1.0) → scale(0.85)  按下收缩
50-150ms → scale(0.85) → scale(1.3)  弹性放大
150-300ms → scale(1.3) → scale(1.0) 回弹到位
```

**技术实现:**
```dart
// 使用 Flutter 的 AnimationController
class LikeButton extends StatefulWidget {
  @override
  Widget build(BuildContext context) {
    return AnimatedBuilder(
      animation: _animation,
      builder: (context, child) {
        return Transform.scale(
          scale: _isPressed ? 0.85 : (_isAnimating ? 1.3 : 1.0),
          child: IconButton(...),
        );
      },
    );
  }
}
```

**动画曲线:**
- 按下: `Curves.easeOut` (50ms)
- 回弹: `Curves.elasticOut` (250ms)

#### 1.3 点赞数展示规则

| 数量范围 | 显示格式 | 示例 |
|----------|----------|------|
| 0 | 不显示数字 | 👍 |
| 1-999 | 完整数字 | 👍 128 |
| 1000-9999 | 1.2k | 👍 1.2k |
| 10000+ | 1w+ | 👍 1w+ |

**数字变化动画:**
- 数字变化时使用 `AnimatedSwitcher`
- 过渡时长: 200ms
- 新数字从上方滑入，旧数字向下方滑出

### P0-3: 真实数据接入 UI 适配

#### 3.1 数据状态映射

| 数据状态 | UI 状态 | 处理方式 |
|----------|---------|----------|
| 加载中 | 骨架屏 | 显示 CommentSkeleton |
| 空数据 | 空状态 | 显示 EmptyComments |
| 有数据 | 评论列表 | 显示 CommentList |
| 加载失败 | 错误状态 | 显示错误提示+重试 |
| 网络异常 | 离线提示 | 显示离线状态+缓存数据 |

#### 3.2 数据结构适配

**评论数据模型 UI 映射:**
```dart
class CommentUIModel {
  // 基础信息
  final String id;
  final String content;
  final DateTime createdAt;
  
  // 用户信息
  final UserInfo user;
  final String userName;
  final String avatarUrl;
  
  // 评分信息
  final double rating;
  final List<String> tags; // 评论标签
  
  // 媒体
  final List<String> imageUrls;
  final List<CommentMedia> media;
  
  // 互动数据
  final int likeCount;
  final bool isLiked;
  final int replyCount;
  final List<CommentReply> replies;
  
  // UI 状态
  final bool isExpanded; // 展开/收起
  final bool isLoading;  // 正在发送
}
```

#### 3.3 图片数据处理

**图片尺寸适配:**
| 图片来源 | 处理方式 | 占位图 |
|----------|----------|--------|
| 网络图片 | CachedNetworkImage | 骨架屏 |
| 本地图片 | Image.file | 灰色占位 |
| 上传中 | 本地预览 + 进度条 | 模糊预览 |

**图片加载状态:**
```
加载前: 灰色占位 (120px x 120px)
加载中: shimmer 效果
加载完成: 图片淡入 (200ms)
加载失败: 错误图标 + 重试按钮
```

### P1-5: 交互动画设计

#### 5.1 点赞动画 (详细)

**动画序列:**
```
帧 0-5:   scale 1.0 → 0.85    (按下)
帧 5-15:  scale 0.85 → 1.3    (弹起+放大)
帧 15-25: scale 1.3 → 1.1     (回弹过冲)
帧 25-30: scale 1.1 → 1.0     (稳定)

同时: 粒子效果从中心向外扩散
```

**粒子效果:**
| 属性 | 数值 |
|------|------|
| 粒子数量 | 8-12个 |
| 粒子形状 | 小圆点/爱心 |
| 粒子颜色 | #2D968A, #FFB800 |
| 扩散半径 | 30px |
| 扩散时长 | 400ms |
| 淡出 | 同时 |

#### 5.2 展开/收起动画

**评论内容展开:**
```dart
AnimatedContainer(
  duration: Duration(milliseconds: 200),
  curve: Curves.easeInOut,
  height: isExpanded ? null : maxHeight,
  child: Text(...),
)
```

| 属性 | 数值 |

#### 5.3 下拉刷新动画

**下拉刷新样式:**
```
下拉中:     释放刷新:    刷新中:
   ↓          ↓           ↏
  ↓↓        ↓↓↓         ↻↻↻
 ↓↓↓       ↓↓↓↓↓       ↻↻↻↻↻
```

**刷新指示器规格:**
| 属性 | 数值 |
|------|------|

| 颜色 | #2D968A |

**下拉进度反馈:**
| 进度 | 视觉反馈 |

| 40-60% | 箭头开始旋转 |

| 80-100% | "释放刷新" 文字出现 |

## 交互说明标注

### 评论列表交互流程

```
用户进入评论列表
      ↓
┌─────────────┐
│ 显示骨架屏  │ ← 300ms 内
└─────────────┘
      ↓
数据加载完成
      ↓
┌─────────────┐     ┌─────────────┐
│ 有数据      │     │ 无数据      │
│ 显示列表    │     │ 显示空状态  │
└─────────────┘     └─────────────┘
      ↓
用户操作:
  - 点击点赞 → 播放动画 + 数字+1
  - 点击回复 → 展开输入框
  - 点击展开 → 展开全文
  - 点击图片 → 进入查看器
  - 下拉列表 → 触发刷新
```

### 点赞交互状态机

```
          ┌─────────────┐
          │   未点赞    │
          │   count=N   │
          └──────┬──────┘
                 │ 点击
                 ↓
          ┌─────────────┐
          │  点赞动画中 │ ← 300ms
          │ scale 1→1.3 │
          └──────┬──────┘
                 │ 动画完成
                 ↓
          ┌─────────────┐
          │   已点赞    │
          │   count=N+1 │
          └──────┬──────┘
                 │ 再次点击
                 ↓
          ┌─────────────┐
          │   未点赞    │
          │   count=N   │
          └─────────────┘
```

### 图片查看器交互

```
进入查看器
    ↓
图片加载中 → 显示 loading
    ↓
图片加载完成 → 淡入显示
    ↓
手势操作:
  - 单击: 切换 UI 显示/隐藏
  - 双击: 放大/还原
  - 捏合: 自由缩放 (1x-3x)
  - 左右滑动: 切换图片
  - 下滑: 退出查看器
```
