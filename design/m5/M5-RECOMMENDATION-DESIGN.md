## 1. 设计概述

### 1.1 设计目标
- 为用户推荐个性化的徒步路线
- 清晰展示推荐理由和匹配度
- 建立用户对推荐系统的信任
- 提供便捷的路线筛选和浏览体验

### 1.2 功能范围

| 功能模块 | 说明 | 优先级 |
|----------|------|--------|
| 推荐列表页 | 推荐结果展示 | P0 |
| 匹配度展示 | 百分比匹配度 | P0 |
| 因子说明 | 推荐理由拆解 | P0 |
| 个性化标签 | 用户画像标签 | P1 |
| 筛选器 | 结果筛选功能 | P1 |

### 1.3 推荐因子定义

| 因子 | 英文名 | 权重 | 图标 | 说明 |
|------|--------|------|------|------|
| 位置距离 | distance | 高 | map-pin | 基于用户位置的距离 |
| 难度匹配 | difficulty | 高 | trending | 匹配用户能力等级 |
| 偏好标签 | preference | 中 | tag | 符合用户喜好标签 |
| 历史行为 | history | 中 | clock | 基于浏览/徒步历史 |
| 季节适宜 | season | 中 | sun | 当前季节的适合度 |
| 热度排名 | popularity | 低 | flame | 路线热门程度 |

## 9. 动效设计

### 9.1 列表加载动效

```
入场动画:
- 触发: 页面进入或刷新
- 动画: 卡片依次淡入上滑
- opacity: 0→1
- translateY: 30px→0
- 时长: 单个 400ms
- 错开: 100ms
- 缓动: ease-out
```

### 9.2 卡片悬停/点击

```
悬停状态 (桌面):
- 阴影增强: shadow-sm → shadow-md
- 轻微上移: translateY(-2px)
- 时长: 200ms
```
