## 1. 文档概述

### 1.1 目标
本文档定义 M5 阶段的设计规范，涵盖新手引导、成就系统、推荐页面三大功能模块的视觉设计、交互设计和设计系统扩展。

### 1.2 适用范围
- M5-DESIGN-SPEC.md - 本文档（规范总览）
- M5-NEWBIE-GUIDE-DESIGN.md - 新手引导详细设计
- M5-ACHIEVEMENT-DESIGN.md - 成就系统详细设计
- M5-RECOMMENDATION-DESIGN.md - 推荐页面详细设计
- assets/badges/ - 徽章 SVG/Lottie 源文件

### 1.3 设计原则

M5 设计延续山径 APP 核心设计原则：

| 维度 | 关键词 | M5 设计表达 |
|------|--------|-------------|
| **自然** | 山野、清新、生机 | 有机曲线、自然纹理、呼吸感动画 |
| **安心** | 可靠、安全、专业 | 清晰的进度反馈、稳定的视觉层级 |
| **诗意** | 雅致、文艺、温度 | 精致徽章设计、温暖的成就表达 |
| **年轻** | 活力、简洁、现代 | 现代动效、个性化推荐 |

## 8. 动效规范

### 8.1 时间函数

| 名称 | 值 | 用途 |
|------|-----|------|
| ease-default | cubic-bezier(0.4, 0, 0.2, 1) | 默认过渡 |
| ease-in | cubic-bezier(0.4, 0, 1, 1) | 元素退出 |
| ease-out | cubic-bezier(0, 0, 0.2, 1) | 元素进入 |
| ease-bounce | cubic-bezier(0.68, -0.55, 0.265, 1.55) | 徽章弹出、庆祝 |
| ease-spring | cubic-bezier(0.175, 0.885, 0.32, 1.275) | 弹性效果 |

### 8.2 动效时长

| 类型 | 时长 | 用途 |
|------|------|------|
| micro | 100ms | 按钮按下、状态切换 |
| fast | 200ms | 悬停效果、小切换 |
| normal | 300ms | 页面切换、卡片展开 |
| slow | 500ms | 引导页过渡、弹窗 |
| dramatic | 800ms | 成就解锁、庆祝动画 |

### 8.3 引导页动效

**页面切换**:
```
- 触发: 滑动/点击下一步
- 动画: 当前页 translateX(-100%), 新页 translateX(0)
- 时长: 400ms
- 缓动: ease-out
```

**插图进入**:
```
- 触发: 页面显示
- 动画: opacity 0→1, scale 0.8→1, translateY(20px)→0
- 时长: 600ms
- 缓动: ease-spring
- 延迟: 200ms (错开文字)
```

**文字进入**:
```
- 触发: 页面显示
- 动画: opacity 0→1, translateY(30px)→0
- 时长: 500ms
- 缓动: ease-out
- 延迟: 0ms
```

**进度点切换**:
```
- 触发: 页面切换
- 动画: width 8px→24px, opacity 变化
- 时长: 300ms
- 缓动: ease-default
```

### 8.4 成就解锁动效

**徽章弹出** (Lottie 动画):
```
阶段1 (0-30%): 徽章 scale 0→1.2, 伴随粒子爆发
阶段2 (30-60%): 徽章 scale 1.2→1, 轻微弹跳
阶段3 (60-100%): 光晕扩散, 文字淡入
总时长: 1500ms
```

**徽章墙加载**:
```
- 触发: 页面进入
- 动画: 每个徽章依次淡入
- 时长: 单个 400ms
- 错开: 50ms 间隔
- 缓动: ease-out
```

## 10. 暗黑模式规范

### 10.1 色彩映射

| 浅色模式 | 深色模式 | 用途 |
|----------|----------|------|
| `#FFFFFF` | `#0F1419` | 页面背景 |
| `#F9FAFB` | `#1A1F24` | 卡片背景 |
| `#111827` | `#FFFFFF` | 主文字 |
| `#6B7280` | `#9CA3AF` | 次要文字 |

### 10.2 成就暗黑模式调整

| 等级 | 浅色主色 | 深色主色 | 发光效果 |
|------|----------|----------|----------|
| 铜 | `#CD7F32` | `#D2691E` | 无 |
| 银 | `#C0C0C0` | `#D3D3D3` | 微光 |
| 金 | `#FFD700` | `#FFC125` | 柔和发光 |
| 钻石 | `#00CED1` | `#00FFFF` | 明显发光 |

### 10.3 引导页暗黑模式

- 插图颜色饱和度降低 20%
- 背景使用深色渐变替代纯色
- 高亮区域使用浅色边框
- 按钮保持品牌主色
