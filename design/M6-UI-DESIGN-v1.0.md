# 山径APP - M6 阶段 UI 设计文档

> **文档版本**: v1.0  
> **制定日期**: 2026-03-20  
> **文档状态**: 高保真设计 - M6  
> **适用范围**: M6 阶段产品设计  
> **设计任务**: 社区功能 UI 设计

---

## 文档说明

本文档为山径APP M6 阶段的UI设计规范，基于设计系统 v1.0 和高保真设计 v1.0，定义社区功能相关页面的视觉细节、精确尺寸、颜色值和交互状态。

**M6 设计范围**:
1. 评论列表和输入框设计
2. 照片瀑布流布局
3. 用户个人主页（带关注按钮）
4. 收藏夹管理页面
5. 照片上传界面

---

## 参考对象

本次设计参考以下产品和设计方向：
- **Instagram** - 照片网格布局、个人主页信息架构
- **小红书** - 瀑布流内容展示、用户互动设计
- **Strava** - 户外社区的专业感和数据展示
- **Apple Photos** - 照片选择的交互体验

### 2.3 瀑布流网格布局

#### 2.3.1 网格规格

| 属性 | 数值 | 说明 |
|------|------|------|
| 列数 | 3列 | 三列瀑布流 |

| 行间距 | 4px | 行与行之间 |

| 单元格宽度 | calc((100% - 16px) / 3) | 自适应宽度 |

#### 2.3.2 单元格高度规则

采用 Pinterest 式瀑布流，每张照片保持原始比例：

| 图片比例 | 高度计算 | 示例 |
|----------|----------|------|
| 竖图 (3:4) | 宽度 * 4/3 | 约 1.33倍宽度 |
| 方图 (1:1) | 宽度 * 1 | 等于宽度 |
| 横图 (4:3) | 宽度 * 3/4 | 约 0.75倍宽度 |
| 宽横图 (16:9) | 宽度 * 9/16 | 约 0.56倍宽度 |

**最小高度**: 120px  
**最大高度**: 300px
