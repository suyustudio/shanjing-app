## 2. 视觉设计

### 2.1 配色方案

**浅色模式卡片**
```
背景: linear-gradient(135deg, #E8F5F3 0%, #B8E0D8 50%, #88CBC0 100%)
或: 图片 + 半透明遮罩 rgba(255,255,255,0.85)

徽章区域背景: #FFFFFF
徽章区域阴影: 0 8px 32px rgba(0,0,0,0.12)

主文字: #1A1A2E
次文字: #4A5568
强调文字: #2D968A
```

**暗黑模式卡片**
```
背景: linear-gradient(135deg, #0A0F14 0%, #1A222C 50%, #2D968A 100%)
或: 深色图片 + 遮罩 rgba(10,15,20,0.9)

徽章区域背景: #141C24
徽章区域阴影: 0 8px 32px rgba(0,0,0,0.3)

主文字: #F0F6FC
次文字: #C9D1D9
强调文字: #4DB6AC
```

## 3. 组件详情

### 3.1 徽章展示区域

| 属性 | 值 |
|------|-----|
| 容器尺寸 | 240×240px |
| 容器形状 | 圆形 |
| 背景 | #FFFFFF (浅色) / #141C24 (暗黑) |
| 阴影 | 0 8px 32px rgba(0,0,0,0.12) |
| 徽章尺寸 | 200×200px |
| 徽章位置 | 居中 |

### 3.2 数据统计区域

```
┌─────────────────────────────────────┐
│                                     │
│   ┌─────────┬─────────┬─────────┐  │
│   │         │         │         │  │
│   │   32    │   156   │   128   │  │
│   │  数字   │  数字   │  数字   │  │
│   │  20px   │  20px   │  20px   │  │
│   │  Bold   │  Bold   │  Bold   │  │
│   │         │         │         │  │
│   │  路线   │  公里   │  小时   │  │
│   │  标签   │  标签   │  标签   │  │
│   │  12px   │  12px   │  12px   │  │
│   │ Regular │ Regular │ Regular │  │
│   │         │         │         │  │
│   └─────────┴─────────┴─────────┘  │
│                                     │
└─────────────────────────────────────┘
```

- 分隔线: 1px solid rgba(0,0,0,0.1)
- 数字颜色: 品牌色 #2D968A / #4DB6AC
- 标签颜色: 次文字色

### 3.3 底部区域

| 元素 | 尺寸 | 说明 |
|------|------|------|
| 二维码 | 80×80px | 指向个人成就页 |
| Logo | 24×24px | 山径品牌Logo |
| 文案 | 12px | "扫码探索更多路线" |
