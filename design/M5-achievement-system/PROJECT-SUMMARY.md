## 设计特点

### 视觉系统
1. **四阶等级体系**: 铜(圆形) → 银(圆形+微光) → 金(圆形+闪烁) → 钻石(六边形+流光)
2. **品牌一致性**: 使用山径品牌色 (Teal 色系)
3. **暗黑模式**: 完整适配浅色/暗黑两种模式
4. **无障碍**: 颜色对比度符合 WCAG AA 标准

### 交互设计
1. **解锁动效**: 弹簧动画 + 光晕扩散 + 彩带效果
2. **进度反馈**: 清晰的进度条和完成百分比
3. **分类筛选**: 支持按类别查看成就
4. **空状态引导**: 提供明确的行动路径

## 使用说明

### 开发人员

1. **徽章使用**: 直接引用 SVG 文件，通过 width/height 控制尺寸
2. **动画集成**: 使用 Lottie Flutter 库加载 JSON 动画文件
3. **主题切换**: 根据系统主题切换对应的颜色 Token

### 设计人员

1. **源文件**: SVG 文件可直接在 Figma/Sketch 中编辑
2. **导出规范**: 如需 PNG 版本，从 SVG 导出 2x/3x 尺寸
3. **扩展设计**: 参考现有徽章风格设计新成就
