## 🔧 开发使用说明

### 徽章使用

```dart
// Flutter 中使用徽章
SvgPicture.asset(
  'assets/badges/first-hike/badge_first_gold_80.svg',
  width: 80,
  height: 80,
)

// 解锁动画
Lottie.asset(
  'assets/lottie/achievement_unlock.json',
  width: 200,
  height: 200,
)
```

### 徽章状态显示

```dart
// 已解锁 - 显示彩色徽章 + ✓ 角标
// 未解锁 - 灰色蒙版 + 🔒 图标
// 新解锁 - 彩色徽章 + "New" 标签 + 发光动画
```

### 暗黑模式适配

所有 SVG 徽章支持通过 CSS 或 Flutter Theme 进行颜色调整。钻石徽章在暗黑模式下会增强流光效果。
