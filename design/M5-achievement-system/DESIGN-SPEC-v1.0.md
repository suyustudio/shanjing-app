## 1. 设计概述

### 1.1 项目背景
山径 App M5 阶段成就系统，通过游戏化激励体系鼓励用户持续探索、记录徒步历程、分享户外体验。

### 1.2 设计目标
- 建立清晰的成就等级体系 (铜/银/金/钻石)
- 设计具有品牌辨识度的徽章视觉系统
- 提供流畅的解锁动效体验
- 支持社交分享的成就卡片

### 1.3 设计原则
| 原则 | 说明 |
|------|------|
| **自然融入** | 徽章设计与山径户外主题一致 |
| **层次分明** | 4级徽章有明显视觉区分 |
| **微光质感** | 高级徽章使用发光效果体现价值感 |
| **暗黑适配** | 全部设计支持浅色/暗黑模式 |

| 等级 | 名称 | 主色 | 边框样式 | 光效 | 形状 |
|------|------|------|----------|------|------|
| 🥉 铜 | Bronze | `#CD7F32` | 1px 实线 | 无 | 圆形 |
| 🥈 银 | Silver | `#C0C0C0` | 2px 双线 | 微光 | 圆形 |

## 3. 成就徽章设计

### 3.1 徽章规格

| 场景 | 尺寸 | 格式 | 用途 |
|------|------|------|------|
| 列表展示 | 80×80px | SVG/PNG | 成就墙网格 |
| 详情展示 | 120×120px | SVG/PNG | 成就详情页 |
| 解锁弹窗 | 160×160px | Lottie | 解锁动画 |
| 分享卡片 | 200×200px | PNG | 分享海报 |

### 3.2 徽章分类设计

#### 类别 1: 首次徒步 (First Hike)
图标概念: 脚印 + 起点旗帜

| 等级 | 徽章名称 | 设计描述 |
|------|----------|----------|
| 铜 | 迈出第一步 | 单个脚印，铜色质感，简单描边 |
| 银 | 探索者 | 双脚印+小旗帜，银色渐变 |
| 金 | 行者无疆 | 脚印路径+飘扬旗帜，金色光芒 |
| 钻石 | 传奇行者 | 脚印 trail + 山顶旗帜，六边形，宝石效果 |

#### 类别 2: 里程累计 (Distance Master)
图标概念: 距离标记 + 路径线

| 等级 | 徽章名称 | 设计描述 |
|------|----------|----------|
| 铜 | 初出茅庐 | 10km 路牌，铜色 |
| 银 | 行路人 | 50km 路牌+路径，银色 |
| 金 | 远行者 | 100km 路牌+山脉，金色 |
| 钻石 | 千里行者 | 500km 路牌+远征路径，钻石 |

#### 类别 3: 路线收集 (Trail Collector)
图标概念: 地图标记 + 路线图标

| 等级 | 徽章名称 | 设计描述 |
|------|----------|----------|
| 铜 | 路线探索者 | 单枚地图标记，铜色 |
| 银 | 路线达人 | 多枚标记+小地图，银色 |
| 金 | 路线专家 | 地图+指南针，金色 |
| 钻石 | 路线大师 | 完整地图+山脉+标记，钻石 |

#### 类别 4: 连续打卡 (Streak Master)
图标概念: 火焰 + 日历/天数

| 等级 | 徽章名称 | 设计描述 |
|------|----------|----------|
| 铜 | 坚持不懈 | 小火苗，铜色 |
| 银 | 习惯养成 | 中火焰+3天标记，银色 |
| 金 | 持之以恒 | 大火焰+30天，金色 |
| 钻石 | 户外狂热 | 火焰 crown + 100天，钻石 |

#### 类别 5: 分享达人 (Social Star)
图标概念: 分享图标 + 星星/爱心

| 等级 | 徽章名称 | 设计描述 |
|------|----------|----------|
| 铜 | 乐于分享 | 分享图标+小星星，铜色 |
| 银 | 内容创作者 | 相机+内容，银色 |
| 金 | 户外博主 | 皇冠+星星，金色 |
| 钻石 | 意见领袖 | 星光 crown + 点赞，钻石 |

### 4.2 组件规范

#### 顶部成就展示卡片
- 背景: `--surface` 或渐变背景
- 徽章尺寸: 120×120px
- 徽章动画: 缓慢旋转/光晕呼吸
- 进度条: 高度 8px, 圆角 4px
- 进度颜色: 品牌色渐变

#### 分类 Tab
- 高度: 40px
- 选中态: 品牌色背景 + 白色文字
- 未选中: 透明背景 + 次要文字色
- 圆角: 20px (pill 形状)

#### 徽章网格
- 列数: 4列 (手机) / 6列 (平板)
- 间距: 16px
- 徽章容器: 80×80px + 标签区域
- 徽章圆角: 12px

#### 徽章状态
| 状态 | 视觉表现 |
|------|----------|
| 已解锁 | 彩色徽章，有 ✓ 角标 |
| 未解锁 | 灰色蒙版，显示 🔒 图标 |
| 新解锁 | 发光动画 + "New" 标签 |

## 5. 成就解锁动效设计

### 5.1 解锁弹窗

```
┌─────────────────────────────────────────┐
│                                         │
│              [黑色遮罩 70%]             │
│                                         │
│         ┌───────────────────┐           │
│         │                   │           │
│         │    [徽章动画]     │           │
│         │      160×160      │           │
│         │    旋转+发光      │           │
│         │                   │           │
│         │   🎉 恭喜解锁！   │           │
│         │                   │           │
│         │   [徽章名称]      │           │
│         │   解锁条件描述    │           │
│         │                   │           │
│         │  [分享]  [确定]   │           │
│         │                   │           │
│         └───────────────────┘           │
│                                         │
│         [Confetti 粒子效果]             │
│                                         │
└─────────────────────────────────────────┘
```

### 5.2 动效时间线

```
0ms     - 遮罩淡入 (opacity 0 → 1, 300ms)
100ms   - 徽章从中心放大 (scale 0 → 1, 500ms, spring)
        - 背景光晕扩散 (opacity 0 → 1 → 0, 800ms)
300ms   - 标题文字淡入 (opacity 0 → 1, 300ms)
400ms   - 描述文字淡入 (opacity 0 → 1, 300ms)
500ms   - 按钮上滑 (translateY 20 → 0, 300ms)
0-1000ms - Confetti 粒子下落
```

### 5.3 徽章动画参数

| 效果 | 参数 | 说明 |
|------|------|------|
| 缩放 | scale 0→1 | 500ms, spring(1, 0.5) |
| 旋转 | rotate 0→360° | 2000ms, linear, loop |
| 光晕 | blur 0→20px | 1000ms, ease-in-out, loop |
| 悬浮 | translateY 0→-10px | 1500ms, ease-in-out, loop |

## 8. 暗黑模式适配

### 8.1 色彩映射

| 元素 | 浅色模式 | 暗黑模式 |
|------|----------|----------|
| 页面背景 | `#FFFFFF` | `#0A0F14` |
