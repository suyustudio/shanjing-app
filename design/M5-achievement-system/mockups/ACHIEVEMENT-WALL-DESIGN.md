# 成就墙页面设计文档

## 1. 页面概述

### 1.1 页面信息
- **页面名称**: 我的成就
- **路由**: `/achievements`
- **功能**: 展示用户所有成就徽章，支持分类筛选和详情查看

### 1.2 设计目标
- 清晰展示成就进度
- 激励用户完成更多成就
- 支持快速分享成就

**容器**
```

内边距: 0 16px
```


**角标**

## 6. 交互设计

### 6.1 点击交互

**徽章点击**
```
按下:
  scale: 0.95
  时长: 100ms

抬起:
  scale: 1
  导航到成就详情页
```

**Tab 切换**
```
切换动画:
  背景色渐变: 200ms
  文字色渐变: 200ms

徽章网格:
  淡入淡出: 150ms
```

### 6.2 加载状态

**页面加载**
```
骨架屏:
  - 卡片区域: 圆角矩形占位
  - Tab 区域: pill 形状占位
  - 徽章网格: 圆形占位 × 8

动画:
  shimmer 效果，从左到右
```

**进度条加载**
```
动画: 从 0% 到实际值
时长: 800ms
缓动: ease-out
```

## 7. Flutter 实现参考

### 7.1 页面结构

```dart
class AchievementWallPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      backgroundColor: Theme.of(context).colorScheme.background,
      appBar: AppBar(
        title: Text('我的成就'),
        actions: [IconButton(icon: Icon(Icons.settings), onPressed: _openSettings)],
      ),
      body: CustomScrollView(
        slivers: [
          // 成就展示卡片
          SliverToBoxAdapter(child: AchievementHeaderCard()),
          
          // 分类筛选
          SliverToBoxAdapter(child: CategoryTabBar()),
          
          // 徽章网格
          SliverPadding(
            padding: EdgeInsets.all(16),
            sliver: AchievementGrid(),
          ),
        ],
      ),
    );
  }
}
```

### 7.2 徽章网格

```dart
class AchievementGrid extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return SliverGrid(
      gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
        crossAxisCount: 4,
        childAspectRatio: 0.85,
        crossAxisSpacing: 16,
        mainAxisSpacing: 16,
      ),
      delegate: SliverChildBuilderDelegate(
        (context, index) => AchievementBadgeItem(
          achievement: achievements[index],
        ),
        childCount: achievements.length,
      ),
    );
  }
}
```
