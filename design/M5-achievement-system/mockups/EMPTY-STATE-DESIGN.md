# 成就系统空状态设计

## 1. 设计概述

### 1.1 场景
- 用户首次进入成就页面，尚未获得任何成就
- 用户筛选某分类，该分类下无成就

### 1.2 设计目标
- 友好引导，不造成挫败感
- 提供明确的行动路径
- 展示可期待的成就预览

## 3. 组件规范

### 3.1 插画区域

| 属性 | 值 |
|------|-----|
| 尺寸 | 200×200px |
| 位置 | 水平居中 |
| 动画 | 叶子飘落、问号微动 |

### 3.2 文字区域

| 元素 | 字体 | 颜色 | 对齐 |
|------|------|------|------|
| 主标题 | 20px, Semibold | --text-primary | center |
| 描述 | 14px, Regular | --text-secondary | center |
| 分隔文字 | 12px, Regular | --text-tertiary | center |

### 3.3 行动按钮

```
样式: 主按钮 (Primary Button)
尺寸: 宽度自适应, 高度 48px
圆角: 12px
文字: 去探索路线, 16px, Medium
图标: 可选, 箭头或指南针
```

### 3.4 推荐卡片

```
容器:
  背景: --surface
  圆角: 12px
  内边距: 16px
  外边距: 8px 水平
  阴影: 0 2px 8px rgba(0,0,0,0.04)

布局:
  左: 徽章缩略图 48×48px
  中: 标题 + 描述
  右: 箭头图标

缩略图:
  尺寸: 48×48px
  圆角: 12px
  背景: --surface-variant
  徽章: 居中, 32×32px

标题:
  16px, Medium, --text-primary

描述:
  12px, Regular, --text-secondary

箭头:
  20px, --text-tertiary
```

## 4. 动画效果

### 4.1 插画动画

**叶子飘落**
```
类型: 循环动画
时长: 3000ms
路径: 从上往下, 带左右摇摆
数量: 3-5片叶子
缓动: ease-in-out
```

**问号微动**
```
类型: 循环动画
时长: 2000ms
动作: 轻微缩放 + 摇摆
scale: 1 → 1.05 → 1
rotate: -5° → 5° → -5°
```

### 4.2 入场动画

**顺序**
```
0ms   - 插画淡入 + 上移
200ms - 标题淡入
300ms - 描述淡入
400ms - 按钮上滑淡入
600ms - 推荐卡片依次淡入
```
