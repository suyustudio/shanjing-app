# 山径APP 地图标记图标设计规范 v1.0

## 设计概述

为山径APP设计一套统一、醒目、适合户外场景的地图标记图标系统。

### 设计目标
- **醒目性**：户外阳光直射下依然清晰可见
- **一致性**：统一的设计语言（圆角、描边、阴影）
- **易识别**：大图标、高对比度、直观符号
- **可操作性**：支持手套操作的大点击区域

| 图标 | 文件名 | 用途 | 主色调 |

## 尺寸规格

### 基础尺寸（@1x）
```
画布尺寸：48 x 48 px
图标主体：40 x 40 px
内边距：4 px
```

### 多倍图规格
| 倍率 | 文件名后缀 | 尺寸 | 用途 |
|------|-----------|------|------|
| @1x | `_mdpi` | 48x48px | 基础尺寸 |
| @2x | `_xhdpi` | 96x96px | 高分辨率屏 |
| @3x | `_xxhdpi` | 144x144px | 超高分辨率 |

### 地图显示尺寸
- **默认显示**：48dp（与系统默认标记相比大 1.5 倍）
- **选中状态**：56dp（放大 1.17 倍）
- **点击区域**：64dp（超出视觉边界，支持手套操作）

## 颜色规范

### 起点标记（绿色系）
```
主色：#4CAF50        // 填充色
深色：#2E7D32        // 描边、阴影
浅色：#81C784        // 高光
文字：#FFFFFF        // 图标内文字
```

### 终点标记（橙红色系）
```
主色：#FF5722        // 填充色（深橙）
深色：#D84315        // 描边、阴影
浅色：#FF8A65        // 高光
文字：#FFFFFF        // 图标内文字
```

### 停车场标记（蓝色系）
```
背景：#2196F3        // 圆形背景
深色：#1565C0        // 描边、阴影
浅色：#64B5F6        // 高光
文字：#FFFFFF        // P字母
```

### 状态颜色
```
正常状态：100% 不透明度
选中状态：添加白色描边（2px）+ 发光效果
悬停状态：轻微放大 + 提升阴影
禁用状态：50% 不透明度
```

## 设计元素规范

### 描边
- **宽度**：2px
- **颜色**：使用各图标的深色值
- **作用**：增强边界清晰度，阳光下可见

### 阴影
- **颜色**：rgba(0, 0, 0, 0.3)
- **偏移**：X: 0, Y: 2px
- **模糊**：4px
- **作用**：提升立体感，增强与地图的层次区分

### 圆角
- **主体圆角**：8px（与设计系统一致）
- **定位针尖**：圆形或尖角设计

### 图标风格
- **类型**：填充风格（solid fill）
- **内图标**：白色线性图标或文字
- **对比度**：确保 WCAG AA 标准

## 图标详细设计

### 1. 起点标记 (marker_start.svg)

```
形状：水滴形/定位针形
┌──────────────┐
│   ┌──────┐   │
│   │ 起   │   │  ← 绿色填充 + 白色"起"字
│   └──────┘   │
│       ▼      │  ← 尖角指向位置
└──────────────┘

尺寸：48x48px
- 圆形主体：36px 直径
- 尖角：8px 高度
- 描边：2px #2E7D32
- 阴影：0 2px 4px rgba(0,0,0,0.3)
```

### 2. 终点标记 (marker_end.svg)

```
形状：水滴形/定位针形（与起点相同形状）
┌──────────────┐
│   ┌──────┐   │
│   │ 终   │   │  ← 橙红色填充 + 白色"终"字
│   └──────┘   │
│       ▼      │  ← 尖角指向位置
└──────────────┘

尺寸：48x48px
- 圆形主体：36px 直径
- 尖角：8px 高度
- 描边：2px #D84315
- 阴影：0 2px 4px rgba(0,0,0,0.3)
```

### 3. 停车场标记 (marker_parking.svg)

```
形状：圆形（国际通用P字标识）
┌──────────────┐
│   ┌──────┐   │
│   │  P   │   │  ← 蓝色圆形 + 白色P字
│   └──────┘   │
│              │
└──────────────┘

尺寸：48x48px
- 圆形背景：40px 直径
- P字大小：24px，字重 Bold
- 描边：2px #1565C0
- 阴影：0 2px 4px rgba(0,0,0,0.3)
```

## 使用场景说明

### 地图缩放层级适配

| 缩放层级 | 图标尺寸 | 显示内容 |
|----------|----------|----------|
| z < 12 | 24px | 简化点状标记 |
| z 12-14 | 36px | 标准图标（无文字） |
| z 15-17 | 48px | 标准图标 + 文字 |
| z > 17 | 56px | 放大图标 + 文字 + 标签 |

### 户外场景优化

1. **高对比度模式**
   - 自动检测环境亮度
   - 强光下增加白色外发光
   - 图标不透明度提升至 100%

2. **夜间模式**
   - 降低亮度避免刺眼
   - 添加柔和发光效果
   - 颜色适当减饱和

3. **雨天/雾天**
   - 增强图标饱和度
   - 增大描边宽度至 3px
   - 提升阴影浓度

## 技术实现要点

### Flutter BitmapDescriptor 生成

```dart
// 从 SVG 生成不同尺寸的 BitmapDescriptor
final markerIcon = await BitmapDescriptor.fromAssetImage(
  const ImageConfiguration(size: Size(48, 48)),
  'assets/markers/marker_start_xhdpi.png',
);
```

### 点击区域配置

```dart
Marker(
  markerId: MarkerId('start'),
  position: position,
  icon: markerIcon,
  anchor: const Offset(0.5, 1.0), // 锚点定位在尖角底部
  // 点击区域由 consumeTapEvents 控制
  consumeTapEvents: true,
)
```
