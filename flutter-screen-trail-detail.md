# M5 Flutter 路线详情页设计

## 页面：路线详情页 (Trail Detail Screen)

### 布局结构
```
┌─────────────────────────┐
│      [封面图]            │  ← 全宽，高度 240px
│                         │
│    [收藏按钮 - 右上角]    │
├─────────────────────────┤
│  路线名称                 │  ← 大标题，24px
│  ⭐⭐⭐ 难度: 中等         │  ← 星级 + 难度标签
├─────────────────────────┤
│  📍 12.5 km              │
│  ⏱️ 约 4 小时             │
├─────────────────────────┤
│  路线简介                 │  ← 小标题，16px
│  这是一条风景优美的徒步    │
│  路线，沿途可欣赏山川湖    │
│  泊，适合周末休闲...      │  ← 正文，14px，3-4行
│                         │
│  [开始导航]               │  ← 底部固定按钮
└─────────────────────────┘
```

### 组件清单

| 组件 | 类型 | 说明 |
|------|------|------|
| CoverImage | Image | 封面图，全宽，圆角 12px |
| FavoriteButton | IconButton | 收藏按钮，心形图标，右上角定位 |
| TrailName | Text | 路线名称，fontSize: 24，bold |
| DifficultyBadge | Chip | 难度标签，颜色区分（绿/黄/红）|
| InfoRow | Row | 距离 + 时长，图标+文字 |
| Description | Text | 简介文本，maxLines: 4 |
| StartButton | ElevatedButton | 底部固定，主色调 |

### 空状态设计

当路线不存在或已下架时显示：

```
┌─────────────────────────┐
│                         │
│                         │
│       [地图图标]         │  ← 灰色图标，64px
│                         │
│  路线不存在或已下架       │  ← 灰色文字，16px
│                         │
│      [返回按钮]          │  ← 主色调边框按钮
│                         │
└─────────────────────────┘
```

| 组件 | 类型 | 说明 |
|------|------|------|
| EmptyIcon | Icon | 地图或路线图标，灰色 #9E9E9E，size: 64 |
| EmptyText | Text | "路线不存在或已下架"，灰色 #757575，fontSize: 16 |
| BackButton | OutlinedButton | "返回"，主色调边框，点击返回上一页 |

### 交互说明
- 点击收藏按钮：切换心形填充状态
- 点击开始导航：跳转导航页
- 封面图：支持左右滑动查看多张图片
- 空状态返回按钮：返回上一页

### 配色建议
- 背景：白色 #FFFFFF
- 主色：品牌青 #2D968A
- 文字：主文字 #212121，次要文字 #757575
- 难度：简单 #2D968A，中等 #FFC107，困难 #F44336
