# 品牌命名 - 人的热情/天才/神性词汇研究

## 核心洞察

用户要的：人的热情、热爱、主动性、创造性、天才、神性

不是AI的能力，是人的内在光芒。

---

## 希腊语词汇

### Daimon / Daimonion
| 属性 | 内容 |
|------|------|
| **原意** | 守护神、内在灵性、天才 |
| **来源** | 苏格拉底称自己的内在声音为 "daimonion" |
| **感觉** | 内在的天才、守护神、灵感 |
| **现代** | 演变为 "demon"（恶魔），但原意是正面的 |
| **变体** | Daimon, Daimonion, Daemon |

**为什么好**：
- 每个人都有内在的 "daimon"（天才/守护神）
- 平台帮助人释放内在的 daimon
- 有哲学深度，像 Midjourney 引用庄子

## 拉丁语词汇

### Numen
| 属性 | 内容 |
|------|------|
| **原意** | 神性、创造力、守护神 |
| **来源** | 罗马神话中的神圣力量 |
| **感觉** | 创造力、神性、灵性 |
| **现代** | Numen = 创造力、天才 |
| **变体** | Numen, Numina |

**为什么好**：
- Numen = 创造力、神性
- 每个人都有 numen（创造力）
- 平台唤醒人的 numen

### Genius（原意）
| 属性 | 内容 |
|------|------|
| **原意** | 守护精灵、创造力、天赋 |
| **来源** | 罗马神话，每个人的守护精灵 |
| **现代** | 演变为 "天才"，但原意更深层 |
| **感觉** | 每个人都有内在的 genius |

**为什么好**：
- 每个人都有 genius（天赋/守护精灵）
- 平台帮助人发现和释放 genius
- 但 "Genius" 已被用烂（Apple Genius Bar）

| **含义** | 内在的天才、守护神、灵感 |

> Daimon 平台帮助创作者释放内在的 genius。

## 变形/组合

| 名称 | 来源 | 感觉 |
|------|------|------|
| **Daimon** | 希腊语原词 | 简洁，有力 |

| **Numina** | 拉丁语复数 | 集体创造力 |
