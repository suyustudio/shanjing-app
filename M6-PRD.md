## 一、功能概述

### 1.1 功能模块
M6 阶段聚焦社交功能建设，包含四大核心模块：

| 模块 | 功能描述 | 优先级 |
|------|----------|--------|
| 评论系统 | 路线评论、评分、回复互动 | P0 |
| 照片分享 | 徒步照片上传、瀑布流展示 | P0 |
| 关注系统 | 用户关注、粉丝管理 | P1 |
| 收藏夹 | 路线分类收藏管理 | P0 |

### 1.2 功能定位
将山径从工具型应用升级为**社区化户外平台**，增强用户粘性和内容沉淀。

### 1.3 核心价值
- **内容沉淀:** 用户产生内容(UGC)丰富路线信息
- **决策辅助:** 真实评价帮助用户选择路线
- **社交激励:** 关注和互动提升用户活跃度
- **个性化管理:** 收藏夹帮助用户规划徒步计划

### 1.4 参考竞品
- **AllTrails:** 评论和照片系统
- **小红书:** 瀑布流和社区互动
- **Strava:** 关注系统和动态流
- **两步路:** 户外社区功能

### 2.5 评分计算规则

| 维度 | 权重 | 说明 |
|------|------|------|
| 总体评分 | 100% | 用户直接给出的1-5星评分 |
| 平均分计算 | - | 去掉最高最低各5%后取平均 |
| 更新时间 | - | 最近30天评价权重×1.2 |
| 认证标识 | - | 完成过该路线的用户标识"体验过" |

### 2.6 评价标签体系

| 类别 | 标签 | 描述 |
|------|------|------|
| 风景 | 风景优美 | 自然景观出色 |
| 风景 | 拍照圣地 | 适合摄影 |
| 风景 | 秋叶绝美 | 秋季推荐 |
| 难度 | 适合新手 | 入门级难度 |
| 难度 | 难度适中 | 需要一定体力 |
| 难度 | 有挑战性 | 较难路线 |
| 设施 | 设施完善 | 休息点/厕所充足 |
| 设施 | 补给方便 | 沿途可购买水/食物 |
| 人群 | 人少安静 | 避世好去处 |
| 人群 | 亲子友好 | 适合带孩子 |

## 六、数据模型

### 6.1 评论数据模型

```typescript
// 评论表 (Review)
interface Review {
  id: string;
  trailId: string;
  userId: string;
  rating: number;           // 1-5
  content: string;          // 最多500字
  difficulty?: string;      // 难度再评估
  tags: string[];           // 标签
  photos: Photo[];          // 照片
  likeCount: number;
  replyCount: number;
  isVerified: boolean;      // 是否体验过
  createdAt: string;
  updatedAt: string;
}

// 评论回复表 (ReviewReply)
interface ReviewReply {
  id: string;
  reviewId: string;
  userId: string;
  parentId?: string;        // 父回复ID（支持嵌套）
  content: string;
  likeCount: number;
  createdAt: string;
}

// 路线评分统计 (TrailRating)
interface TrailRating {
  trailId: string;
  average: number;          // 平均分
  count: number;            // 评价数
  distribution: {           // 分布
    5: number;
    4: number;
    3: number;
    2: number;
    1: number;
  };
  tags: {                   // 标签统计
    name: string;
    count: number;
  }[];
}
```

### 6.2 照片数据模型

```typescript
// 照片表 (Photo)
interface Photo {
  id: string;
  userId: string;
  trailId?: string;         // 关联路线（可选）
  location?: {
    lat: number;
    lng: number;
    name?: string;
  };
  url: string;
  thumbnailUrl: string;
  description?: string;
  width: number;
  height: number;
  likeCount: number;
  commentCount: number;
  isPublic: boolean;
  createdAt: string;
}

// 照片点赞表 (PhotoLike)
interface PhotoLike {
  photoId: string;
  userId: string;
  createdAt: string;
}
```

interface Follow {
