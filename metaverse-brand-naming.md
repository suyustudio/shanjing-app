# 「元宇宙UGC平台」品牌命名方案

## 项目核心要素

| 维度 | 关键信息 |
|------|----------|
| **产品定位** | 元宇宙UGC平台，AI Agent驱动创作 |
| **核心差异** | H5自主引擎、AI辅助创作、房间式社交 |
| **目标用户** | 15-24岁创作型玩家（探索者、爱好者、准专业者） |
| **目标市场** | 东南亚（印度/印尼/菲律宾/越南）第一梯队 |
| **竞品对标** | Roblox、YAHAHA、BUD |
| **技术特色** | 浏览器即开即玩、AI Agent、组件化创作 |

---

## 命名方向分析

基于项目特点，命名需要传达：
1. **创作/创造** - UGC核心
2. **空间/世界** - 元宇宙属性
3. **AI/智能** - 技术差异
4. **年轻/活力** - 目标用户
5. **全球/无国界** - 海外市场

## 与竞品对比

| 竞品 | 名称特点 | 我们差异化 |
|------|----------|------------|
| **Roblox** | 造词，无意义 | 有意义，有文化底蕴 |
| **YAHAHA** | 语气词，活泼 | 更雅致，更专业 |
| **BUD** | 单字，简洁 | 可单字可双字，更灵活 |
| **Meta** | 直接，宏大 | 更具体，更创作导向 |
