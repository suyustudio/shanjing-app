|---------|------|
| **事件触发** | 4个P0事件在正确时机触发 |

## 4. 数据上报链路验证

### 4.1 上报链路检查

```
客户端触发事件
    ↓
Analytics SDK 组装数据
    ↓
网络请求上报（10秒超时）
    ↓
├─ 成功 → 数据进入分析平台
└─ 失败 → 缓存本地，等待重试
    ↓
网络恢复后批量补发
```

### 4.2 链路验证清单

| 检查项 | 验证方法 | 验证状态 |
|--------|---------|---------|
| 事件正常上报 | 抓包查看 HTTP 请求 | ⬜ |
| 上报成功率 > 95% | 对比触发次数和上报次数 | ⬜ |
| 超时时间 10秒 | 模拟慢网环境验证 | ⬜ |
| 失败重试机制 | 断网后恢复验证补发 | ⬜ |
| 数据格式正确 | 后端接收日志检查 | ⬜ |
| 时间戳准确 | 对比客户端和服务端时间 | ⬜ |

### 4.3 数据质量检查

| 检查项 | 标准 | 验证状态 |
|--------|------|---------|
| 必填参数完整率 | 100% | ⬜ |
| 参数类型正确率 | 100% | ⬜ |
| 枚举值合法率 | 100% | ⬜ |
| 时间戳格式正确 | 毫秒级时间戳 | ⬜ |
| 坐标精度合理 | 精度 > 0 且 < 100 | ⬜ |
