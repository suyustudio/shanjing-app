## 1. 概述

### 1.1 设计目标
本文档定义山径APP中**安全类POI**的视觉设计规范，确保用户在户外徒步时能够快速识别安全设施，在紧急情况下获得清晰、醒目的视觉指引。

### 1.2 安全类POI范围
| POI类型 | 核心功能 | 紧急程度 |
|---------|----------|----------|
| **紧急避险点** | 恶劣天气、身体不适时的避难所 | 高 |
| **信号覆盖区** | 需要联系外界时的通信点 | 高 |
| **下撤点** | 无法完成全程时的安全退出点 | 高 |
| **医疗点** | 受伤或突发疾病时的救助点 | 极高 |
| **SOS求救点** | 配备救援设备的固定点位 | 最高 |

### 1.3 设计原则
1. **最高辨识度**：安全类POI必须在地图上第一眼被识别
2. **警示色系**：使用红色/橙色等高对比度警示色
3. **紧急感传达**：通过视觉元素传递紧迫、重要的感受
4. **离线优先**：在无网络环境下依然清晰可见
5. **防误触设计**：SOS等关键功能需防止误操作

### 2.3 色彩使用规范

#### 2.3.1 图标色彩
- **图标填充色**：使用对应POI的主色
- **图标描边**：白色2px描边，确保在深色地图上可见
- **选中状态**：主色加深10% + 外发光效果

#### 2.3.2 背景色彩
- **正常状态**：白色背景 + 主色图标
- **选中状态**：浅色系背景（如安全红浅 `#FFEBEE`）+ 深色图标
- **紧急模式**：深色背景 + 高对比度图标

#### 2.3.3 文字色彩
- **POI名称**：深灰 `#1A1A1A`
- **距离信息**：中灰 `#666666`
- **紧急提示**：对应主色
- **SOS按钮文字**：白色 `#FFFFFF`

## 3. 图标设计规范

### 3.1 图标尺寸规范

| 场景 | 尺寸 | 说明 |
|------|------|------|
| **地图默认** | 28×28dp | 比普通POI大20% |
| **地图选中** | 36×36dp | 放大1.3倍 |
| **紧急模式** | 48×48dp | SOS专用超大尺寸 |
| **卡片图标** | 24×24dp | 信息卡片内 |
| **列表图标** | 20×20dp | POI列表内 |

### 3.2 图标设计原则

#### 3.2.1 通用规范
- **风格**：面性图标 + 白色细节
- **线条粗细**：2px
- **圆角**：2px（柔和友好）
- **视觉重量**：保持一致的视觉重量感
- **可识别性**：户外强光下3米外可识别

## 9. 总结

### 9.1 核心设计要点

1. **最高优先级**：安全类POI在视觉上始终处于最高层级
2. **警示色系**：红色/橙色为主，确保高辨识度
3. **尺寸优势**：安全POI图标比普通POI大20%，SOS更大
4. **离线优先**：所有设计考虑无网络环境下的可用性
5. **防误触设计**：SOS功能需要长按3秒确认
