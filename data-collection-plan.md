# P9 数据采集计划

## 1. 数据来源

| 来源类型 | 说明 | 优先级 |
|---------|------|--------|
| UGC | 用户上传跑步/骑行轨迹数据 | P0 |
| 爬虫 | 公开地图平台 POI 数据 | P1 |
| 合作 | 与运动 App/设备厂商数据合作 | P2 |

---

## 2. 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 名称 | string | 路线/地点名称 |
| 距离 | number(km) | 路线总长度 |
| 时长 | number(min) | 预估完成时间 |
| 坐标 | [lng, lat] | 起点/终点/途经点 |
| 类型 | enum | 跑步/骑行/徒步 |
| 难度 | int(1-5) | 综合难度评级 |
| 标签 | string[] | 风景/夜跑/爬坡等 |

---

## 3. 质量审核流程

```
数据入库 → 自动校验 → 人工抽检 → 用户反馈 → 定期清理
```

- **自动校验**: 坐标合法性、距离/时长合理性
- **人工抽检**: 10% 抽样审核
- **用户反馈**: 举报机制 + 评分系统
- **定期清理**: 30天无互动数据下架
