## 6. 动画规格

### 6.1 转场动画

| 转场 | 类型 | 时长 | 缓动函数 |
|------|------|------|----------|
| 页面进入 | 从右滑入 | 300ms | cubic-bezier(0.4, 0, 0.2, 1) |
| 页面退出 | 向右滑出 | 250ms | cubic-bezier(0.4, 0, 1, 1) |
| 弹窗进入 | 从底部滑入 + 淡入 | 250ms | cubic-bezier(0, 0, 0.2, 1) |
| 弹窗退出 | 向下滑出 + 淡出 | 200ms | cubic-bezier(0.4, 0, 1, 1) |
| 按钮按下 | 缩放 | 100ms | ease-out |
| 按钮释放 | 弹性恢复 | 200ms | cubic-bezier(0.175, 0.885, 0.32, 1.275) |

### 6.2 持续动画

| 动画 | 时长 | 循环 | 说明 |
|------|------|------|------|
| 录制指示器 | 1500ms | 无限 | 呼吸灯效果 |
| 当前位置脉冲 | 2000ms | 无限 | 扩散波纹 |
| GPS信号搜索 | 1000ms | 无限 | 闪烁效果 |
| 轨迹绘制 | 根据长度 | 一次 | 实时绘制 |

### 6.3 微交互动画

```css
/* 按钮按下 */
.btn:active {
  transform: scale(0.92);
  transition: transform 100ms ease-out;
}

/* 按钮释放 */
.btn {
  transition: transform 200ms cubic-bezier(0.175, 0.885, 0.32, 1.275);
}

/* 录制指示器呼吸 */
@keyframes recording-breathe {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.4; }
}
.recording-indicator {
  animation: recording-breathe 1.5s ease-in-out infinite;
}

/* 位置脉冲 */
@keyframes location-pulse {
  0% { transform: scale(1); opacity: 1; }
  100% { transform: scale(2.5); opacity: 0; }
}
.location-pulse::after {
  animation: location-pulse 2s ease-out infinite;
}
```
