# 山景产品术语表

> 本文档统一山景产品中的术语使用规范，确保产品、设计、开发、运营等团队沟通一致性。

---

## 一、核心术语对照表

| 推荐术语 | 不推荐/避免使用 | 说明 |
|---------|---------------|------|
| **离线包** | 离线地图包 | 简称为"离线包"，指下载到本地的地图数据包 |
| **路线** | 轨迹 | 用户规划或记录的行径路径称为"路线" |
| **导航** | 跟随 | 指系统引导用户沿路线行进的功能 |
| **起点** | 出发点、起始点 | 路线的开始位置 |
| **终点** | 目的地、到达点 | 路线的结束位置 |
| **途经点** | 途径点、中间点 | 路线中经过的中间位置 |
| **当前位置** | 我的位置、所在位置 | 用户当前的GPS定位点 |
| **地图模式** | 视图模式 | 地图的显示样式（如卫星图、地形图等） |
| **缩放级别** | 地图层级 | 地图的放大/缩小级别 |
| **图层** | 覆盖层 | 地图上叠加的信息层 |
| **收藏** | 收藏夹、我的地点 | 用户保存的地点或路线 |
| **搜索** | 查找、查询 | 查找地点、路线等功能 |
| **分享** | 发送、导出 | 将路线或地点分享给他人 |
| **记录** | 录制、追踪 | 记录用户行进路线 |
| **导入** | 加载、读取 | 将外部数据导入应用 |
| **导出** | 保存、下载 | 将数据导出到外部 |
| **海拔** | 高度、高程 | 地点的海拔高度 |
| **坡度** | 倾斜度 | 路线的陡峭程度 |
| **里程** | 距离、长度 | 路线的总长度 |
| **预计时间** | 耗时、需要时间 | 完成路线预计所需时间 |
| **偏航** | 偏离路线 | 用户偏离规划路线 |
| **重新规划** | 重新计算、重新导航 | 根据当前位置重新计算路线 |
| **语音播报** | 语音提示、语音导航 | 导航时的语音指引 |
| **模拟导航** | 演示导航、预览导航 | 不实际移动，预览导航过程 |
| **实景导航** | AR导航、实景引导 | 结合摄像头实景的导航方式 |

## 五、术语使用原则

1. **一致性**：同一概念在全产品中使用统一术语
2. **简洁性**：优先使用简短、易懂的词汇
3. **准确性**：术语应准确表达功能含义
4. **用户友好**：避免使用过于专业的技术术语
5. **场景适配**：根据使用场景选择最合适的表达
