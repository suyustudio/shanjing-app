## 1. 设计概述

### 1.1 M3 设计目标

M3 阶段聚焦**用户系统**和**社交分享**两大核心模块，为山径APP建立完整的用户闭环：

| 模块 | 设计目标 | 用户体验目标 |
|------|----------|--------------|
| **登录/注册** | 极简流程，降低门槛 | 3步完成，<30秒 |
| **分享海报** | 精美模板，一键生成 | 激发分享欲望 |
| **主题切换** | 无缝切换，沉浸体验 | 跟随系统/手动切换 |
| **SOS** | 醒目可靠，防误触 | 紧急时刻零思考 |
| **用户中心** | 信息聚合，操作便捷 | 一站式个人管理 |

### 1.2 设计原则

延续 M2 设计系统，强调：

1. **安全感优先** - 所有用户操作有明确反馈
2. **减少认知负荷** - 界面简洁，信息层级清晰
3. **情感化设计** - 分享海报有温度，有美感
4. **无障碍支持** - 表单验证、错误提示清晰
