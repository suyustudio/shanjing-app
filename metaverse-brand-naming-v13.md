# 「元宇宙UGC平台」品牌命名 - 第十三版（现代数字化动词）

## 避开的词

| 词 | 为什么不能用 |
|----|--------------|
| Craft | 手工感，Minecraft/Roblox |
| Forge | 原始工业感 |
| Cast/Mold | 工业制造 |
| Build | 太通用 |
| Make | 太通用 |

**需要：现代、数字化、AI时代的"制作"动词**

---

## 现代数字化动词

| 词 | 含义 | 感觉 |
|----|------|------|
| **Spin** | 旋转/生成 | 快速，数字 |
| **Render** | 渲染 | 数字，视觉 |
| **Synthesize** | 合成 | AI，科技 |
| **Generate** | 生成 | AI，直接 |
| **Compose** | 组合 | 创作，艺术 |
| **Assemble** | 组装 | 组件化 |
| **Stream** | 流动 | 实时，连续 |
| **Flow** | 流动 | 顺畅，自然 |
| **Pulse** | 脉冲 | 节奏，活力 |
| **Spark** | 火花 | 点燃，灵感 |
| **Bloom** | 绽放 | 生长，有机 |
| **Branch** | 分支 | 生长，扩展 |
| **Mesh** | 网格 | 连接，网络 |
| **Node** | 节点 | 网络，连接 |
| **Link** | 连接 | 关联，网络 |
