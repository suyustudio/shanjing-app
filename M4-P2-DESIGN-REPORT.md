## 1. 字体层级完善

### 1.1 设计决策

**选用 Material Design 3 字体层级系统**

理由：
- 与 Flutter Material 3 组件库天然契合
- 五级层级（Display/Headline/Title/Body/Label）覆盖所有场景
- 每级三种规格（Large/Medium/Small）提供精细粒度控制

### 1.2 字体层级规范

| 层级 | Large | Medium | Small | 典型用途 |
|------|-------|--------|-------|----------|
| **Display** | 57px/Regular | 45px/Regular | 36px/Regular | 启动页、Hero区域 |
| **Headline** | 32px/Regular | 28px/Regular | 24px/Regular | 页面/模态框标题 |
| **Title** | 22px/Medium | 16px/Medium | 14px/Medium | 导航栏、卡片标题 |
| **Body** | 16px/Regular | 14px/Regular | 12px/Regular | 正文、描述、辅助 |
| **Label** | 14px/Medium | 12px/Medium | 11px/Medium | 按钮、标签、徽章 |

### 1.3 设计亮点

- **Display 层级**：从 57px 递进到 36px，满足大标题的视觉冲击力
- **Title 层级**：使用 Medium 字重，在页面层级中建立清晰的视觉焦点
- **Body 层级**：Regular 字重保证长文本阅读舒适度
- **Label 层级**：Medium 字重确保按钮和标签的可读性
- **字间距优化**：Display Large 使用负字间距（-0.25px）增强紧凑感

## 2. 圆角层级规范

### 2.1 设计决策

**从单一圆角扩展为六级圆角系统**

原问题：仅有 8px 单一圆角，无法区分组件重要性和视觉层次。

解决方案：建立 0px → 4px → 8px → 12px → 16px → 999px 的完整圆角梯度。

### 2.2 圆角层级规范

| Token | 值 | 使用场景 | 视觉特点 |
|-------|-----|----------|----------|
| `--radius-none` | 0px | 分割线、列表、表格 | 锐利专业 |
| `--radius-sm` | 4px | 小按钮、标签、徽章 | 精致小巧 |
| `--radius-md` | 8px | 卡片、输入框、主按钮 | 标准平衡 |
| `--radius-lg` | 12px | 大卡片、图片容器、弹窗 | 柔和友好 |
| `--radius-xl` | 16px | 底部弹窗、侧边面板 | 现代感强 |
| `--radius-full` | 999px | 胶囊按钮、搜索栏、头像 | 完全圆润 |

### 2.3 设计亮点

- **层次分明**：从锐利到圆润的渐进变化，适应不同组件需求
- **品牌一致性**：12px 作为"大圆角"主值，延续山径APP自然友好的品牌调性
- **实用性强**：999px 全圆角覆盖所有胶囊形状需求
- **易于扩展**：4px 步长便于未来增加中间值

### 2.4 组合原则

```
外层容器圆角 ≥ 内层元素圆角

示例：
- 图片卡片 (12px) > 图片 (8px)
- 底部弹窗 (16px) > 内容卡片 (8px)
```

## 3. 导航栏滚动效果设计

### 3.1 设计目标

解决页面滚动时导航栏的视觉突兀问题，通过平滑过渡增强：
- 空间感与沉浸体验
- 信息层级的渐进展示
- 品牌调性的细腻表达

### 3.2 效果定义

| 状态 | 触发条件 | 核心变化 |
|------|----------|----------|
| **初始状态** | 滚动位置 = 0 | 透明背景，大标题（20px），无阴影 |
| **滚动中** | 0 < 滚动位置 < 100 | 背景渐显（0% → 95%），标题渐小（20px → 17px） |
| **固定状态** | 滚动位置 ≥ 100 | 实色背景（95%），毛玻璃效果，底部阴影 |

### 3.3 动画参数

| 属性 | 初始值 | 固定值 | 过渡时长 | 缓动曲线 |
|------|--------|--------|----------|----------|
| 背景透明度 | 0% | 95% | 100ms | ease-out |
| 毛玻璃模糊 | 0px | 20px | 100ms | ease-in-out |
| 标题字号 | 20px | 17px | 200ms | ease-out |
| 阴影强度 | 无 | 8px/30% | 100ms | ease-out |
| 底边框 | 透明 | #21262D | 100ms | linear |
| 导航栏高度 | 56px | 48px | 100ms | ease-in-out |

### 3.4 设计亮点

- **渐进式过渡**：100px 滚动距离内完成所有属性变化，避免突兀感
- **毛玻璃效果**：20px 模糊半径营造 iOS 风格的现代感
- **高度压缩**：56px → 48px 的轻微压缩增加内容展示空间
- **临界值设计**：100px 作为临界点，兼顾视觉变化和操作便捷

### 3.5 暗黑模式适配

- 背景色：`#0A0F14` with 95% 透明度
- 底边框：`#21262D`
- 标题色：`#F0F6FC`
- 阴影：纯黑 30% 透明度

已提供完整的 Flutter ScrollableAppBar 组件和 CSS 实现，位于 `design-system-dark-mode-v1.0.md` 第3.6节。
