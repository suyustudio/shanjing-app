# P12 边界场景：GPS信号弱

## 场景：GPS信号弱或丢失

### 触发条件
- 进入室内/地下/隧道等信号盲区
- 设备定位权限被关闭
- 网络连接中断导致无法获取位置

### 用户提示文案

| 场景 | 提示文案 |
|------|----------|
| GPS信号弱 | "定位信号较弱，正在使用最后已知位置" |
| GPS信号丢失 | "无法获取当前位置，请检查定位权限或手动刷新" |
| 位置过期 | "位置信息已过期（超过30分钟），建议手动刷新" |

### 降级方案

1. **使用最后已知位置**
   - 缓存用户最近一次有效定位
   - 显示位置时标注"约 X 分钟前"
   - 最大缓存时间：30分钟

2. **提示手动刷新**
   - 提供"刷新位置"按钮
   - 点击后尝试重新获取GPS定位
   - 失败时提示检查权限或网络

### 交互示意

```
┌─────────────────────────────┐
│  ⚠️ 定位信号较弱              │
│  正在使用 5 分钟前的位置      │
│                             │
│  [刷新位置]  [继续使用]       │
└─────────────────────────────┘
```
