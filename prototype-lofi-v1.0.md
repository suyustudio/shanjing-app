# 山径APP - 低保真原型设计文档（第1部分）

> **文档版本**: v1.0  
> **制定日期**: 2026-02-27  
> **文档状态**: 设计阶段  
> **适用范围**: Week 2 产品设计阶段 - P4-1  
> **设计范围**: 启动页/引导页、首页-发现列表、路线详情页（基础信息部分）

---

## 文档说明

本文档为山径APP的低保真原型设计，基于信息架构v1.0和用户流程图v1.0设计，聚焦核心页面的结构、布局和基础交互。

**低保真原型特点**:
- 使用文字描述和ASCII线框图表示页面结构
- 不关注视觉细节（颜色、字体、圆角等）
- 重点表达信息层级、布局结构和交互逻辑


## 5. 通用交互规范

### 5.1 页面转场规范

| 场景 | 过渡方式 | 时长 | 说明 |
|------|----------|------|------|
| 一级→二级页面 | 从右滑入 | 300ms | 标准页面跳转 |
| 二级→一级页面 | 从右滑出 | 300ms | 返回操作 |
| 弹层出现 | 从底部滑入 | 250ms | 筛选器/分享等 |
| 弹层消失 | 向下滑出 | 200ms | 关闭弹层 |
| 模态弹窗 | 淡入+缩放 | 200ms | 提示/确认框 |
| Tab切换 | 无/淡入 | 150ms | 同层级切换 |

### 5.2 加载状态规范

| 场景 | 加载方式 | 说明 |
|------|----------|------|
| 页面初始化 | 全屏Loading | 骨架屏或转圈 |
| 列表加载更多 | 底部Loading | 上拉触发 |
| 下拉刷新 | 顶部Loading | 下拉触发 |
| 按钮提交 | 按钮Loading | 禁用+转圈 |
| 图片加载 | 占位图+淡入 | 渐进显示 |

### 5.3 反馈规范

| 场景 | 反馈方式 | 时长 |
|------|----------|------|
| 操作成功 | Toast提示 | 2秒 |
| 操作失败 | Toast提示+震动 | 2.5秒 |
| 网络错误 | 页面提示+重试按钮 | 常驻 |
| 重要确认 | 弹窗确认 | 需用户操作 |
