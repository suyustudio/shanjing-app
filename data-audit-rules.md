# P11 数据审核流程 - 自动校验规则

## 校验规则

| 规则类型 | 校验内容 | 校验逻辑 |
|---------|---------|---------|
| 必填字段 | 名称、类型、坐标、图片 | 字段为空或 null 时判定失败 |
| 坐标范围 | 经度、纬度 | 经度: -180 ~ 180, 纬度: -90 ~ 90 |
| 图片尺寸 | 宽度、高度 | 最小: 100x100px, 最大: 8000x8000px |

## 校验失败处理

| 场景 | 处理方式 |
|-----|---------|
| 必填字段缺失 | 标记为"待补充", 通知提交人补全 |
| 坐标超出范围 | 拒绝并提示用户重新输入 |
| 图片尺寸不符 | 拒绝入库, 返回具体错误信息 |

## 流程图

```
数据提交 → 自动校验 → 全部通过 → 入库
              ↓
         校验失败 → 按规则处理 → 重新提交
```

---

## 状态流转图

```
待审核 → 审核中 → 已通过
            ↘
              已拒绝
```

- **待审核**：数据提交后等待审核的初始状态
- **审核中**：审核人员正在处理的状态
- **已通过**：审核通过，数据正式入库
- **已拒绝**：审核不通过，退回修改或拒绝入库

# P13 数据审核流程 - 人工抽检

## 抽检规则

| 项目 | 规则 |
|-----|------|
| 抽检比例 | 10% |
