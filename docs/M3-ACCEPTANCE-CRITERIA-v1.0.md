## 2. 验收检查项

### 2.1 用户系统

- [ ] 微信登录2秒内调起，授权后3秒内完成
- [ ] 手机号验证码60秒倒计时正常
- [ ] Token过期自动刷新，用户无感知
- [ ] 首次启动必须同意隐私政策
- [ ] 支持添加/编辑/删除紧急联系人
- [ ] 导航前检查并引导设置联系人

### 2.2 离线地图

- [ ] Android离线地图可下载并使用
- [ ] 断网后自动切换离线地图
- [ ] 下载进度实时显示
- [ ] iOS离线地图可用（P1）

### 2.3 分享功能

- [ ] 3秒内生成海报，图片<2MB
- [ ] 短链接100%可访问
- [ ] H5落地页3秒内首屏渲染
- [ ] 已安装APP时直接唤起

### 2.4 SOS功能

- [ ] 长按3秒触发，有震动反馈
- [ ] 二次确认弹窗防止误触
- [ ] 10秒内发送短信给紧急联系人
- [ ] 位置精度<50米
- [ ] 激活后每分钟更新位置
- [ ] 30秒内可取消

### 2.5 测试覆盖

- [ ] 单元测试覆盖率>80%
- [ ] 核心功能E2E测试通过
- [ ] 实地测试完成（九溪/龙井/宝石山）
- [ ] GPS弱信号场景测试通过
- [ ] 低电量场景测试通过
