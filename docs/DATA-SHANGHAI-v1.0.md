## 二、路线详情

### 路线1：古美公园环湖轻徒步

#### 基本信息
- **位置**: 上海市闵行区平吉路799号（平吉路与合川路交叉口东南角）
- **坐标范围**: 31.160°N ~ 31.163°N, 121.394°E ~ 121.397°E
- **海拔**: 约5米（全程平坦）
- **门票**: 免费

#### 路线特色
古美公园（又称古美体育公园）以蝶形湖泊为中心，四周环绕绿植、湿地、亲水平台和木栈道。公园布局紧凑、形态丰富，移步换景，生趣盎然。

**主要景点**:
- 蝶形湖泊 - 公园核心景观
- 亲水平台 - 近距离观赏湖景
- 木栈道 - 环湖而行，穿梭于绿植湿地
- 健身步道 - 专业锻炼场地
- 休憩凉亭 - 多处纳凉休息点

#### 实用信息
| 项目 | 详情 |
|------|------|
| 停车场 | 公园南门附近，约50个车位 |
| 卫生间 | 南门入口东侧 |
| 补给 | 入口自动售货机 |
| 地铁 | 12号线虹梅路站 |
| 公交 | 平吉路合川路站 |

#### 适合人群
- 全年龄段适宜
- 特别适合亲子家庭、老年人晨练
- 宠物友好（需牵绳）

#### 最佳时间
- 清晨 6:00-8:00（观鸟最佳）
- 傍晚 17:00-19:00（赏落日）
- 春季、秋季气候最宜人

### 路线2：漕河泾港滨水步道

#### 基本信息
- **位置**: 上海徐汇区龙华西路至龙吴路段
- **坐标范围**: 31.175°N ~ 31.183°N, 121.446°E ~ 121.463°E
- **海拔**: 约8米（全程平坦）
- **门票**: 免费

#### 路线背景
漕河泾港是徐汇区"百里河"工程的骨干河道，全长7.36公里。本次采集的路段（龙华西路至龙吴路）全长约3.6公里，于2020-2021年完成两岸贯通整治，是徐汇区水环境治理的典范。

**治理亮点**:
- 采用分级过滤、双膜引流、浮岛生态净化等技术
- 水质清澈度稳定达1米以上
- 新增滨水绿化5.5万平方米
- 累计贯通提升18.6公里

#### 路线特色
- 生态浮岛 - 鸢尾、菖蒲等水生植物净化水质
- 清澈河道 - 可见鱼儿游弋，水质优良
- 塑胶跑道 - 专业健身设施，配有距离标识
- 滨水绿化 - 四季常青的河岸景观

#### 实用信息
| 项目 | 详情 |
|------|------|
| 停车场 | 龙华西路路边停车/漕河泾开发区配套 |
| 卫生间 | 沿线多个节点设有公厕 |
| 补给 | 沿线众多便利店、咖啡厅 |
| 地铁 | 12号线龙漕路站（5分钟步行） |
| 公交 | 多条线路可达沿线 |

#### 适合人群
- 全年龄段适宜
- 特别适合上班族午间锻炼
- 无障碍通道，轮椅、婴儿车可通行

#### 最佳时间
- 全天皆宜
- 早晨 6:00-9:00
- 下午 17:00-19:00

### 坐标数据说明
- 坐标系: WGS84
- 精度: 6位小数（约0.1米精度）
- 采集方式: 基于地图位置估算
- 轨迹点间距: 50-100米

### 数据局限性
1. 坐标数据为估算值，实际徒步时需以现场路径为准
2. 公园设施信息可能随时间变化
3. 停车场容量为估算值
4. 建议实地验证后再发布
