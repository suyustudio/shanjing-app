# M3 实地导航测试计划

**版本**: v1.0.0+3 (Build #35)  
**日期**: 2026-03-14  
**测试类型**: 实地户外测试  
**测试地点**: 杭州西湖环山区域

---

## 一、测试目标

1. 验证GPS定位精度和稳定性
2. 验证导航语音播报准确性
3. 验证偏航检测和重新规划功能
4. 验证离线导航能力
5. 评估电量消耗表现

---

## 三、测试场景

### 3.1 正常导航测试

| 测试项 | 测试步骤 | 预期结果 | 通过标准 |
|--------|----------|----------|----------|
| 起点定位 | 打开APP，进入导航页 | GPS定位在起点附近(<10米) | 定位成功 |
| 路线规划 | 选择测试路线，开始导航 | 显示规划路线和预计时间 | 路线显示正确 |
| 语音播报 | 跟随路线行走 | 在关键节点听到语音指引 | 播报及时准确 |
| 轨迹记录 | 全程保持导航开启 | 轨迹线与实际路径吻合 | 轨迹准确 |
| 到达终点 | 走到路线终点 | 显示到达提示 | 正确识别终点 |

### 3.2 偏航检测测试

| 测试项 | 测试步骤 | 预期结果 | 通过标准 |
|--------|----------|----------|----------|
| 小偏航 | 偏离路线10米内 | 无反应或轻微提示 | 不触发重规划 |
| 中度偏航 | 偏离路线20-50米 | 提示"已偏离路线" | 有语音提示 |
| 严重偏航 | 偏离路线>50米 | 自动重新规划路线 | 生成新路线 |
| 偏航恢复 | 返回原路线 | 恢复正常导航 | 无需重新规划 |
| 多偏航 | 连续多次偏航 | 每次都能正确检测 | 无遗漏 |

### 3.3 离线导航测试

| 测试项 | 测试步骤 | 预期结果 | 通过标准 |
|--------|----------|----------|----------|
| 离线启动 | 关闭网络，打开APP | 提示离线模式，功能可用 | 正常进入 |
| 离线地图显示 | 无网络下查看地图 | 显示已下载的离线地图 | 地图正常 |
| 离线导航 | 关闭网络开始导航 | 导航功能正常 | 无网络可用 |
| 离线偏航 | 离线状态下偏航 | 偏航检测和重规划正常 | 功能正常 |
| 网络恢复 | 恢复网络连接 | 自动切换在线模式 | 无缝切换 |

### 3.4 GPS精度测试

| 测试项 | 测试步骤 | 预期结果 | 通过标准 |
|--------|----------|----------|----------|
| 开阔地带 | 在广场/湖面测试 | GPS精度<5米 | 高精度 |
| 树木遮挡 | 在树林路段测试 | GPS精度<10米 | 可接受 |
| 建筑遮挡 | 靠近建筑物测试 | GPS精度<15米 | 可接受 |
| 隧道/桥下 | 经过隧道或桥下 | 使用最后已知位置 | 不丢失 |
| GPS漂移 | 静止时观察 | 位置不大幅漂移 | 稳定 |

## 四、电量消耗测试

### 4.1 测试方案

| 测试项 | 方法 | 测量指标 |
|--------|------|----------|
| 基准电量 | 记录起始电量 | 百分比 |
| 每小时耗电 | 每小时记录一次 | %/小时 |
| 全程耗电 | 记录结束电量 | 总消耗 |
| 后台耗电 | 切换后台运行 | 后台消耗 |
| 屏幕耗电 | 对比亮屏/熄屏 | 屏幕影响 |

### 4.2 耗电基准参考

| 场景 | 可接受耗电 | 良好耗电 | 优秀耗电 |
|------|------------|----------|----------|
| 1小时导航 | <15% | <12% | <10% |
| 3小时徒步 | <40% | <35% | <30% |
| 后台保持 | <5%/小时 | <3%/小时 | <2%/小时 |

## 十、通过标准

### 10.1 必须满足（P0）

- [ ] GPS定位可用（精度<15米）
- [ ] 导航功能正常（可完整走完路线）
- [ ] 语音播报准确（无遗漏、无错误）
- [ ] 偏航检测有效（能检测并提示）
- [ ] 耗电可接受（3小时<40%）

### 10.2 应该满足（P1）

- [ ] GPS精度良好（开阔地带<5米）
- [ ] 离线导航可用
- [ ] 偏航重规划正常
- [ ] 耗电良好（3小时<35%）

### 10.3 期望满足（P2）

- [ ] GPS精度优秀（开阔地带<3米）
- [ ] 耗电优秀（3小时<30%）
- [ ] 多设备一致性良好
