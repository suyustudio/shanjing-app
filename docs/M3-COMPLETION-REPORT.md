## 3. 关键设计决策

### 3.1 用户系统

| 决策 | 选择 | 理由 |
|------|------|------|
| 首选登录方式 | 微信登录 | 转化率高，用户体验好 |
| Token机制 | 双Token | 安全+体验平衡 |
| Token有效期 | Access 2h / Refresh 7d | 兼顾安全和便利 |
| 紧急联系人 | 首次导航引导 | 提高设置率 |

### 3.2 分享功能

| 决策 | 选择 | 理由 |
|------|------|------|
| 海报生成 | 客户端生成 | 减少服务器压力 |
| 实时分享 | 支持导航中分享 | 增强安全感 |
| 分享追踪 | UTM参数 | 精准追踪转化 |
| 平台适配 | 动态模板 | 各平台最佳体验 |

### 3.3 SOS功能

| 决策 | 选择 | 理由 |
|------|------|------|
| 触发方式 | 长按3秒 | 防误触+快速触发平衡 |
| 发送通道 | 短信+推送 | 提高送达率 |
| 位置更新 | 每分钟 | 精度与电量平衡 |
| 有效期 | 2小时 | 紧急救援时间窗口 |
