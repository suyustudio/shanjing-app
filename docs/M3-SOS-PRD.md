## 1. SOS功能概述

### 1.1 目标
为山径APP用户提供可靠的紧急求救功能，在用户遭遇危险时能够快速、准确地将位置和求救信息发送给紧急联系人，最大程度保障用户安全。

### 1.2 设计原则

| 原则 | 说明 | 优先级 |
|------|------|--------|
| **简单快速** | 2秒内完成SOS触发 | P0 |
| **可靠送达** | 多渠道发送，确保消息送达 | P0 |
| **信息准确** | 包含精确位置、时间、路线信息 | P0 |
| **误触防护** | 二次确认+取消机制 | P0 |
| **隐私保护** | 位置信息仅用于紧急求救 | P0 |

### 1.3 SOS使用场景

| 场景 | 触发原因 | 期望响应 |
|------|----------|----------|
| **身体不适** | 徒步中受伤或突发疾病 | 紧急联系人收到位置，可安排救援 |
| **迷路失联** | 偏离路线无法返回 | 紧急联系人获取最后已知位置 |
| **遭遇危险** | 野生动物、恶劣天气等 | 快速通知多人，提高获救概率 |
| **协助他人** | 遇到其他需要帮助的徒步者 | 同时通知自己的紧急联系人 |

## 2. SOS触发流程

### 2.1 触发入口

SOS按钮位于导航页面，设计原则：**显眼但不突兀，易于操作但防止误触**

```
┌─────────────────────────────────┐
│                                 │
│     [地图区域]                   │
│                                 │
│                                 │

│  📍 距下一个路口 50m            │
│  ┌─────┐ ┌─────┐ ┌─────┐       │
│  │暂停  │ │结束  │ │ 🆘  │       │ ← SOS按钮
│  └─────┘ └─────┘ └─────┘       │
```

- 位置: 底部按钮栏右侧

- 大小: 56x56dp (触摸友好)

## 5. 隐私和安全考虑

### 5.1 数据隐私保护

#### 位置数据使用原则

| 原则 | 说明 |
|------|------|
| **最小化收集** | 仅收集SOS触发时的必要位置信息 |
| **目的限制** | 位置数据仅用于紧急求救，不作他用 |
| **时效限制** | SOS结束后位置数据不再更新，链接失效 |
| **用户控制** | 用户可随时取消SOS，停止位置共享 |

#### 位置数据生命周期

```
SOS触发
    │
    ▼
```

### 5.2 安全措施

#### 防止滥用

| 措施 | 实现方式 |
|------|----------|
| **触发限制** | 同一用户24小时内最多触发3次SOS |
| **冷却时间** | 每次SOS间隔至少5分钟 |
| **验证机制** | 首次SOS需完成实名认证（可选） |
| **行为分析** | 异常频繁触发标记为可疑账号 |

#### 误触处理

```
SOS触发
    │
    ▼
┌─────────────────┐
│ 二次确认弹窗    │ ← 防止误触
│ (5秒自动确认)   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 发送前再次确认  │ ← 最后防线
│ [确认发送]      │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 显示取消按钮    │ ← 随时可取消
│ (发送后30秒内)  │
└─────────────────┘
```

#### 取消机制

| 取消时机 | 操作 | 效果 |
|----------|------|------|
| 确认前 | 点击"取消" | 不发送任何信息 |
| 发送后30秒内 | 点击"取消发送" | 停止后续发送，已发送的无法撤回 |
| SOS激活后 | 点击"解除求救" | 停止位置更新，通知联系人已安全 |

## 6. 技术实现要点

### 6.1 客户端实现

```dart
class SOSService {
  // 触发SOS
  Future<SOSResult> triggerSOS() async {
    // 1. 检查权限
    if (!await _checkPermissions()) {
      return SOSResult.failure('权限不足');
    }
    
    // 2. 获取位置
    final location = await _getCurrentLocation(timeout: Duration(seconds: 5));
    
    // 3. 收集信息
    final sosData = SOSData(
      userId: currentUser.id,
      location: location,
      timestamp: DateTime.now(),
      routeId: navigationService.currentRoute?.id,
      battery: await _getBatteryLevel(),
    );
    
    // 4. 上报服务器
    final result = await apiService.sendSOS(sosData);
    
    // 5. 启动位置更新（如成功）
    if (result.success) {
      _startLocationUpdates(result.shareToken);
    }
    
    return result;
  }
  
  // 持续位置更新
  void _startLocationUpdates(String shareToken) {
    Timer.periodic(Duration(minutes: 1), (timer) async {
      if (!_isSOSActive) {
        timer.cancel();
        return;
      }
      
      final location = await _getCurrentLocation();
      await apiService.updateSOSLocation(shareToken, location);
    });
  }
  
  // 取消SOS
  Future<void> cancelSOS() async {
    _isSOSActive = false;
    await apiService.cancelSOS();
    Analytics.trackEvent('sos_cancelled');
  }
}
```

### 6.2 后端实现

```typescript
// SOS服务
@Injectable()
class SOSService {
  async triggerSOS(userId: string, data: SOSData): Promise<SOSResult> {
    // 1. 生成分享token
    const shareToken = this.generateToken();
    
    // 2. 获取紧急联系人
    const contacts = await this.userService.getEmergencyContacts(userId);
    
    // 3. 逆地理编码
    const address = await this.geoService.reverseGeocode(data.location);
    
    // 4. 生成消息内容
    const message = this.generateSOSMessage(data, address);
    
    // 5. 发送通知（并行）
    const sendPromises = [
      this.sendSMS(contacts, message),
      this.sendPushNotification(contacts, data),
    ];
    
    const results = await Promise.allSettled(sendPromises);
    
    // 6. 记录日志
    await this.logSOSEvent(userId, data, results);
    
    return {
      success: true,
      shareToken,
      sendResults: results,
    };
  }
  
  // 位置更新
  async updateLocation(shareToken: string, location: Location) {
    // 验证token有效性
    const sosSession = await this.getSOSSession(shareToken);
    if (!sosSession || sosSession.isExpired()) {
      throw new Error('SOS session expired');
    }
    
    // 更新位置
    await this.cache.set(`sos:location:${shareToken}`, location, { ttl: 7200 });
    
    // 通知前端（WebSocket）
    this.wsGateway.broadcastLocationUpdate(shareToken, location);
  }
}
```
