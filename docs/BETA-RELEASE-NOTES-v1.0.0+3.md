## 四、测试重点

### 4.1 实地导航测试

请优先测试以下功能：

1. **GPS定位精度** - 在开阔地带、树林、建筑附近分别测试
2. **导航语音** - 检查播报时机和内容准确性
3. **偏航检测** - 故意偏离路线，观察检测和重规划
4. **离线导航** - 关闭网络后测试导航功能
5. **电量消耗** - 记录1-3小时导航的电量消耗

### 4.2 用户系统测试

1. **注册流程** - 手机号注册是否顺畅
2. **登录方式** - 密码登录和验证码登录
3. **状态保持** - 关闭APP后是否保持登录
4. **微信登录** - 一键登录体验

### 4.3 性能测试

1. **启动速度** - 冷启动和热启动感受
2. **流畅度** - 页面切换、地图缩放是否流畅
3. **内存占用** - 长时间使用是否卡顿
