## 模块说明

### 左侧：待审核列表
| 元素 | 说明 |
|------|------|
| 路线卡片 | 显示路线名称 + 提交时间 |
| 选中状态 | 当前选中卡片高亮显示 |
| 滚动区域 | 列表超出时支持滚动 |
| 数量提示 | 顶部显示"待审核：X条" |

### 右侧：审核详情
| 区域 | 内容 |
|------|------|
| 图片预览 | 路线封面图/多张图片轮播 |
| 信息面板 | 名称、时间、提交人、起终点、里程、难度、标签 |
| 路线描述 | 文字描述内容 |
| 地图预览 | 路线轨迹地图（可选） |
| 操作按钮 | 通过 / 拒绝 |

## 交互说明

1. **点击左侧列表项** → 右侧加载对应路线详情
2. **点击通过** → 标记审核通过，从列表移除，自动加载下一条
3. **点击拒绝** → 弹出拒绝原因输入框，确认后从列表移除
4. **空状态** → 列表为空时显示"暂无待审核路线"

## 极简原则

- 无复杂筛选，按提交时间倒序排列
- 无批量操作，逐条审核
- 无侧边栏导航，专注审核任务
- 通过/拒绝后立即切换到下一条，高效处理
