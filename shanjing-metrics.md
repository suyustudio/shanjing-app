# 北极星指标定义

## 日活跃用户数 (DAU - Daily Active Users)

### 定义

**日活跃用户数**是指在**自然日**（00:00-23:59）期间，至少完成一次**核心业务行为**的独立用户数量。

### 核心行为标准

用户需完成以下任一行为方可计入当日活跃用户：

| 行为类型 | 说明 | 触发条件 |
|---------|------|---------|
| 内容消费 | 浏览/阅读/观看核心内容 | 单次停留 ≥ 10 秒 |
| 内容互动 | 点赞/评论/分享/收藏 | 成功提交互动动作 |
| 内容创作 | 发布/上传内容 | 成功发布并可见 |
| 交易行为 | 完成核心交易流程 | 订单提交成功 |

### 计算方式

```
DAU = 当日完成核心行为的独立用户数（去重）
```

#### 计算规则

1. **去重原则**：同一用户当日多次活跃只计 1 次
2. **跨端去重**：同一账号在不同设备登录，只计 1 次
3. **时间窗口**：以自然日 00:00-23:59 为统计周期
4. **识别方式**：优先使用用户 ID，无登录用户使用设备 ID（需标注为"未登录用户"）

#### 计算公式

```
DAU = COUNT(DISTINCT user_id) 
      WHERE event_time >= '当日00:00' 
        AND event_time < '次日00:00'
        AND event_type IN ('核心行为列表')
```

### 数据口径

| 口径类型 | 说明 | 使用场景 |
|---------|------|---------|
| 全量 DAU | 包含登录用户 + 未登录访客 | 整体产品活跃度 |
| 登录 DAU | 仅统计已登录用户 | 核心用户粘性分析 |
| 新增 DAU | 当日首次完成核心行为的新用户 | 新用户转化效果 |
| 回流 DAU | 连续 7 天未活跃后当日重新活跃的用户 | 用户召回效果 |

### 监控目标

- **当前基准**：待补充
- **目标值**：待补充
- **预警阈值**：单日环比下降 ≥ 10% 触发预警
