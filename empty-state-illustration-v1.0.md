## 1. 设计概述

### 1.1 设计目标

为空状态和异常状态设计一套统一、有品牌识别度的插画系统：

| 目标 | 说明 | 实现方式 |
|------|------|----------|
| **情感安抚** | 减轻用户遇到问题的焦虑感 | 温和的造型、柔和的色彩 |
| **品牌传达** | 传递山径自然、诗意的品牌调性 | 自然元素、山野意象 |
| **指引清晰** | 配合文案提供明确的操作指引 | 恰当的隐喻、直观的图形 |
| **技术可行** | 确保开发可实现、性能可接受 | 矢量优先、适度复杂度 |

### 1.2 设计理念

**"山野间的温柔陪伴"** - 即使在空状态，也让用户感受到山径的温度：

- **自然元素** - 使用山、树、云、路径等山野元素
- **柔和线条** - 避免尖锐的边角，使用有机曲线
- **温暖配色** - 使用品牌色系，营造舒适感
- **适度留白** - 保持简洁，不拥挤

### 1.3 状态分类

| 大类 | 具体状态 | 优先级 | 适用场景 |
|------|----------|--------|----------|
| **网络异常** | 无网络连接 | P0 | 离线状态 |
| **数据为空** | 无数据 | P0 | 列表为空、搜索结果为空 |
| **加载失败** | 加载出错 | P0 | 请求失败、超时 |
| **权限相关** | 无定位权限 | P1 | 权限被拒绝 |
| **功能相关** | 暂无收藏 | P1 | 收藏为空 |
| **功能相关** | 暂无历史 | P1 | 历史记录为空 |
| **搜索相关** | 搜索无结果 | P1 | 搜索为空 |

## 2. 插画风格指南

### 2.1 风格定义

#### 整体风格

| 属性 | 定义 | 说明 |
|------|------|------|
| **艺术风格** | 扁平插画 + 微质感 | 现代简洁，不失温度 |
| **线条风格** | 无描边 / 1px 细描边 | 根据场景选择 |
| **色彩风格** | 品牌色 + 中性灰 | 保持品牌一致性 |
| **透视风格** | 2.5D / 等距视角 | 增加立体感 |
| **比例风格** | 夸张比例 | 头部稍大，增加亲和力 |

## 7. 插画技术规范

### 7.1 文件格式

| 用途 | 格式 | 说明 |
|------|------|------|
| **设计源文件** | Figma / Sketch / AI | 保留可编辑性 |
| **开发交付** | SVG | 矢量格式，可缩放 |
| **备用格式** | PNG @1x, @2x, @3x | 兼容旧系统 |
| **动画版本** | Lottie (JSON) | 如需动画效果 |

### 7.2 尺寸规范

#### 基础尺寸

| 密度 | 尺寸 | 用途 |
|------|------|------|
| **@1x** | 200×200px | 标准显示 |
| **@2x** | 400×400px | 高清屏幕 |
| **@3x** | 600×600px | 超高清 |

#### 实际显示尺寸

| 场景 | 显示尺寸 | 说明 |
|------|----------|------|
| **空状态页面** | 120×120dp | 居中显示 |
| **小空状态** | 80×80dp | 列表内空状态 |
| **Toast提示** | 48×48dp | 小图标形式 |

### 7.3 SVG 规范

#### 导出设置

```xml
<!-- 标准SVG结构 -->
<svg width="200" height="200" viewBox="0 0 200 200" 
     xmlns="http://www.w3.org/2000/svg"
     fill="none">
  
  <!-- 定义颜色变量，便于暗黑模式适配 -->
  <defs>
    <color id="primary" value="#2D968A"/>
    <color id="secondary" value="#72C0B5"/>
    <color id="accent" value="#FFC107"/>
    <color id="neutral" value="#E5E7EB"/>
  </defs>
  
  <!-- 插画内容 -->
  <g id="illustration">
    <!-- ... -->
  </g>
  
</svg>
```

#### 优化要求

| 检查项 | 要求 | 工具 |
|--------|------|------|
| **文件大小** | < 20KB | SVGO |
| **图层命名** | 语义化命名 | 手动检查 |
| **颜色使用** | 使用定义的颜色变量 | 手动检查 |
| **路径简化** | 删除冗余锚点 | Illustrator |
| **兼容性** | 支持移动端渲染 | 多设备测试 |

### 7.4 Lottie 动画规范

如需动画版本：

| 参数 | 值 | 说明 |
|------|-----|------|
| **帧率** | 30fps | 平衡流畅与性能 |
| **时长** | 2-3秒 | 循环播放 |
| **文件大小** | < 50KB | 控制体积 |
| **动画类型** | 循环循环 | 持续播放 |

## 9. 暗黑模式适配

### 9.1 颜色映射

| 浅色模式 | 暗黑模式 | 用途 |
|----------|----------|------|
| `#2D968A` | `#4DB6AC` | 主色元素 |
| `#72C0B5` | `#6BC4BC` | 辅助元素 |
| `#9DD6CE` | `#8DD4CE` | 浅色装饰 |
| `#E5E7EB` | `#30363D` | 中性色背景 |
| `#FFFFFF` | `#141C24` | 白色背景 |
| `#EF5350` | `#F87171` | 错误色 |
| `#FFC107` | `#FFD54F` | 警告/点缀 |

### 9.2 调整原则

1. **提高明度** - 暗黑模式下插画整体明度提高 10-15%
2. **增加对比** - 确保与深色背景有足够对比
3. **避免纯白** - 白色元素改为浅灰 `#C9D1D9`
4. **发光效果** - 可适当添加柔和发光增加层次感
