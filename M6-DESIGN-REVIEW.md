## 四、建议优化项

### 4.1 组件化建议

建议将评论相关 UI 抽取为独立组件：

```
lib/
├── widgets/
│   └── social/                    # 新增社交组件目录
│       ├── comment_list.dart      # 评论列表
│       ├── comment_item.dart      # 单条评论
│       ├── comment_input.dart     # 评论输入框
│       ├── rating_bar.dart        # 评分条
│       ├── review_overview.dart   # 评分统计
│       ├── photo_grid.dart        # 照片网格
│       └── user_avatar.dart       # 用户头像
```

### 4.2 代码规范建议

1. **使用设计系统常量**:
   ```dart
   // 不推荐
   TextStyle(fontSize: 14, fontWeight: FontWeight.w500)
   
   // 推荐
   TextStyle(
     fontSize: DesignSystem.fontBody,
     fontWeight: DesignSystem.weightMedium,
   )
   ```

2. **颜色使用**:
   ```dart
   // 不推荐
   Color(0xFF2D968A)
   
   // 推荐
   DesignSystem.getPrimary(context)
   ```

3. **间距使用**:
   ```dart
   // 不推荐
   SizedBox(height: 16)
   
   // 推荐
   SizedBox(height: DesignSystem.spacingLarge)
   ```

### 4.3 性能优化建议

1. **评论列表优化**:
   - 使用 `ListView.builder` 替代 `Column`
   - 添加 `AutomaticKeepAliveClientMixin`
   - 使用 `cached_network_image` 缓存头像

2. **图片加载优化**:
   - 使用缩略图 + 原图模式
   - 预加载相邻图片
   - 使用 `FadeInImage` 实现淡入效果

### 4.4 无障碍建议

1. 添加语义标签:
   ```dart
   Semantics(
     label: '用户${userName}的评论：${content}',
     child: CommentItem(...),
   )
   ```

2. 确保足够的点击区域 (最小 44x44)

3. 支持屏幕阅读器
