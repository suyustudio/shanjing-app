# 品牌命名方案 - 第二版

## 反馈调整

**上版问题：**
- "野路"谐音"夜路"（负面联想）
- "野"字显得不够高级

**调整方向：**
- 保留自然/户外感，但提升精致度
- 强调"轻量探索"而非"荒野探险"
- 向"高级感+自然疗愈"靠拢

---

## 新方案一：意象转化派 — 「山径」

| 维度 | 说明 |
|------|------|
| **命名逻辑** | "山"（自然/户外）+ "径"（小径/路径） |
| **核心联想** | 山中小径，曲径通幽，精致探索 |
| **情感价值** | 诗意、静谧、发现之美 |
| **词源** | "远上寒山石径斜"——杜牧《山行》 |

**高级感来源：**
- "径"字比"路"更雅致，有"曲径通幽"的文化意象
- 单字"山"+单字"径"，简洁有力
- 古诗词出处，自带文化底蕴

**延展性：**
- 品牌口号："每条山径，都有风景"
- 功能模块：山径指南、山径集、山径印记
- 会员身份：山径行者

**听觉测试：** 2音节，平仄相间（平-仄），朗朗上口

---

## 新方案二：复合造词派 — 「轻山」

| 维度 | 说明 |
|------|------|
| **命名逻辑** | "轻"（轻量/轻松）+ "山"（自然/户外） |
| **核心联想** | 轻量级的山行体验，无负担的自然亲近 |
| **情感价值** | 轻松、自由、无压力 |
| **差异化** | 直接传递"轻度"定位，与硬核户外区隔 |

**高级感来源：**
- "轻"字自带精致感（轻奢、轻食、轻旅行）
- 2字组合简洁现代，符合互联网产品调性
- 发音清脆（qīng shān），有辨识度

**延展性：**
- 品牌口号："上轻山，轻上山"
- 功能模块：轻山指数、轻山搭子、轻山日记
- 视觉方向：极简、留白、呼吸感

**听觉测试：** 2音节，叠韵（ang-ang），好记好传播

## 新方案三：文化语言派 — 「步云」

| 维度 | 说明 |
|------|------|
| **命名逻辑** | "步"（徒步）+ "云"（云端/自在） |
| **核心联想** | 漫步云端，身轻如燕，精神逃离 |
| **情感价值** | 超脱、自由、精神快充 |
| **词源** | "行到水穷处，坐看云起时"——王维 |

**高级感来源：**
- "云"字有禅意和诗意，提升品牌调性
- 古诗词意象，文化底蕴深厚
- 暗示" elevation"（海拔提升）和" elevation of mood"（心情提升）

**延展性：**
- 品牌口号："每一步，都在云上"
- 功能模块：步云记录、步云心情、步云社区
- 会员体系：步云者 → 云中客 → 云游家

**听觉测试：** 2音节，平仄相间（仄-平），抑扬顿挫

## 新方案四：科技隐喻派 — 「途里」

| 维度 | 说明 |
|------|------|
| **命名逻辑** | "途"（旅途）+ "里"（里程/里面） |
| **核心联想** | 在旅途中，也在自然里 |
| **情感价值** | 沉浸、归属、身心在场 |
| **发音特点** | 谐音"土里"，接地气但不土气 |

**高级感来源：**
- 双关语义：既是"旅途"也是"自然里"
- 简洁现代，2音节易传播
- 有"归途""故里"的温暖感

**延展性：**
- 品牌口号："每一步，都在途里"
- 功能模块：途里指南、途里圈、途里印记
- 视觉方向：温暖、人文、生活感

**听觉测试：** 2音节，叠韵（u-i），顺口好记

## 新方案五：意象转化派 — 「麓行」

| 维度 | 说明 |
|------|------|
| **命名逻辑** | "麓"（山脚）+ "行"（行走） |
| **核心联想** | 山麓漫步，轻松可达的自然 |
| **情感价值** | 亲近、可达、无门槛 |
| **词源** | "麓"指山脚，比"山"更具体，暗示轻度 |

**高级感来源：**
- "麓"字雅致不常见，有辨识度但不生僻
- 专业户外术语（山麓），暗示专业性
- 2字组合有节奏感

**延展性：**
- 品牌口号："从麓行，向山行"
- 功能模块：麓行指南、麓行圈、麓行记录
- 会员体系：麓行者 → 山行者 → 峰行者

**听觉测试：** 2音节，平仄相间（仄-平），朗朗上口

## 推荐排序

| 排名 | 方案 | 核心优势 | 潜在风险 |
|------|------|----------|----------|
| **1** | **山径** | 诗意+高级感+文化底蕴 | "径"字稍文气 |
| **2** | **轻山** | 现代感+轻度定位清晰 | 可能过于轻量 |
| **3** | **步云** | 禅意+精神价值强 | "云"字互联网泛化 |
| **4** | **途里** | 温暖+双关语义 | "里"字地域感 |
| **5** | **麓行** | 专业感+辨识度 | "麓"字认知门槛稍高 |

## 首推方案：「山径」

**理由：**
1. **高级感** - "径"字雅致，古诗词出处
2. **自然感** - 直接关联山行场景
3. **轻度暗示** - "径"比"路"小，比"道"轻
4. **听觉优秀** - 2音节，平仄相间，好记好传播
5. **延展性强** - 可构建完整的品牌语言体系

**品牌语言体系预览：**
- 行前："今天走哪条山径？"
- 行中："山径导航，不迷路"
- 行后："又发现一条好山径"
- 社交："山径搭子"
- 内容："山径集"（精选路线）

**视觉方向：**
- 色彩：青绿（山）+ 米白（径）+ 墨黑（文字）
- 字体：现代宋体或人文无衬线体
- 图形：抽象山径线条、等高线、脚印
- 摄影：诗意山景、人在自然中的剪影

---

## 备选方案：「轻山」

如果希望更强调"轻量"定位，"轻山"是强备选：
- 更直接的轻度暗示
- 更现代的互联网调性
- "轻"字符合当下消费趋势

**对比：**
| 维度 | 山径 | 轻山 |
|------|------|------|
| 调性 | 诗意、人文 | 现代、轻快 |
| 定位 | 发现之美 | 轻松无负担 |
| 用户感知 | 探索者 | 享受者 |
| 视觉风格 | 东方美学 | 极简现代 |
