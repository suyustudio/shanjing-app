- 导致状态流转不协调，用户体验可能受影响

1. **修改 `MockNaviState.navigating` 处理逻辑**:

   - 从偏航状态恢复时调用 `_recalculateRoute()`
2. **修改 `MockNaviState.offRoute` 处理逻辑**:
   - 移除立即的 `_recalculateRoute()` 调用
   - 只设置状态和语音提示，等待模拟服务3秒后恢复

- 行373-391: 扩展 `MockNaviState.navigating` 处理，增加 `offRoute` 恢复逻辑
- 行402-410: 修改 `MockNaviState.offRoute` 处理，移除立即重新规划
- 状态流转: `offRoute` → (3秒) → `MockNaviState.navigating` → `_recalculateRoute()` → `navigatingRoute`
