### 2. 遇到的全部问题及解决方案

#### 问题 1: Android embedding V1/V2 冲突
**错误**: `This app is using a deprecated version of the Android embedding.`
**原因**: `flutter pub get` 在 Android 项目不存在时，自动创建默认 V1 项目
**解决**: 使用 `flutter create` 创建项目，然后修改 `AndroidManifest.xml` 添加 V2 embedding

#### 问题 2: 包名非法
**错误**: `"shanjing-app" is not a valid Dart package name.`
**原因**: `flutter create` 不接受带 `-` 的目录名
**解决**: 使用 `--project-name=shanjing_app` 参数

#### 问题 3: Kotlin 版本不兼容
**错误**: `Your project requires a newer version of the Kotlin Gradle plugin.`
**原因**: Flutter 3.19 使用新的 Gradle Plugin DSL，Kotlin 版本在 `settings.gradle` 中定义
**解决**: 修改 `settings.gradle` 中的 `kotlin-android` 版本从 `1.7.10` 到 `1.9.10`

#### 问题 4: 缺少 .env 文件
**错误**: `No file or variants found for asset: .env`
**原因**: `.env` 在 `.gitignore` 中，CI 环境没有此文件
**解决**: 在 CI 中创建 `.env` 文件

#### 问题 5: minSdkVersion 不兼容
**错误**: `minSdkVersion 19 cannot be smaller than version 21 declared in library [:flutter_tts]`
**原因**: `flutter create` 创建的项目使用 `flutter.minSdkVersion`（值为 19），但 `flutter_tts` 需要 21
**解决**: 替换 `flutter.minSdkVersion` 为硬编码 `21`

#### 问题 6: permission_handler API 变化
**错误**: `The getter 'isGranted' isn't defined for the class 'PermissionStatus'`
**原因**: permission_handler 11.0+ 改变了 API，从属性访问变为枚举比较
**解决**: 
- 旧: `locationStatus.isGranted`
- 新: `locationStatus == PermissionStatus.granted`

#### 问题 7: 高德地图 SDK API 变化
**错误**: `AMapLocationOption`, `AMapLocationMode`, `GeoLanguage` 未定义
**原因**: 高德地图 SDK 3.0+ 改变了 API
**解决**: 简化定位代码，移除高级配置选项

#### 问题 8: AMapWidget 参数变化
**错误**: `No named parameter with the name 'myLocationEnabled'`
**原因**: `amap_flutter_map` 3.0+ 移除了此参数
**解决**: 移除 `myLocationEnabled` 和 `myLocationStyle` 参数

#### 问题 9: EdgeInsets 参数问题
**错误**: `No named parameter with the name 'bottom'`
**原因**: `EdgeInsets.symmetric` 在某些 Flutter 版本中不支持 `bottom`
**解决**: 使用 `EdgeInsets.only(left: 16, right: 16, bottom: 8)`

#### 问题 10: 缺少 import
**错误**: `Type 'StreamSubscription' not found`
**原因**: 缺少 `dart:async` import
**解决**: 添加 `import 'dart:async';`

#### 问题 11: 缺少定位权限
**错误**: App 启动 crash，手动开启权限后可运行
**原因**: `flutter create` 创建的 `AndroidManifest.xml` 没有定位权限
**解决**: 添加 `ACCESS_FINE_LOCATION`, `ACCESS_COARSE_LOCATION` 等权限

#### 问题 12: 高德地图 SO 库缺失 ⏳
**错误**: `java.lang.NoClassDefFoundError: Failed resolution of: Lcom/amap/api/maps/AMapOptions;`
**原因**: `flutter create` 创建的 build.gradle 没有配置 abiFilters，SO 库未打包
**解决**: 添加 `ndk { abiFilters "arm64-v8a", "armeabi-v7a", "x86_64" }`

#### 问题 13: 高德地图 Key 缺失 ⏳
**错误**: 启动 crash，可能是 Key 未配置
**原因**: `flutter create` 创建的 `AndroidManifest.xml` 没有高德 Key
**解决**: 添加 `<meta-data android:name="com.amap.api.v2.apikey" android:value="..."/>`
