### 2. 发现的问题

#### 高优先级
1. **MapScreen AppBar 渐变未适配暗黑模式** - 硬编码使用亮色主色
2. **AppCard 亮色背景硬编码** - 使用 `Colors.white` 而非设计系统常量
3. **AppButton 小尺寸圆角不一致** - 小按钮使用 4px，中/大使用 8px
4. **AppButton 次按钮背景硬编码** - 使用 `Colors.white`

#### 中优先级
5. 字体层级过于简单，缺少 display/headline/title/label 尺寸
6. 部分组件仍有硬编码颜色

#### 低优先级
7. 缺少 Skeleton 骨架屏统一使用
8. 缺少主题切换开关

### 3. 已修复问题

提交了以下修复:

1. **MapScreen 渐变 AppBar 暗黑模式适配**
   ```dart
   colors: Theme.of(context).brightness == Brightness.dark
       ? [DesignSystem.backgroundSecondaryDark, DesignSystem.backgroundDark]
       : [DesignSystem.primary.withOpacity(0.9), DesignSystem.primary.withOpacity(0.7)]
   ```

2. **AppCard 背景色改用设计系统常量**
   - `Colors.white` → `DesignSystem.background`

3. **AppButton 圆角统一**
   - 所有尺寸统一使用 `DesignSystem.radius` (8px)

4. **AppButton 次按钮背景**
   - `Colors.white` → `DesignSystem.background`
