## 技术决策

### 1. 空安全处理策略
使用 `??` 提供默认值，避免运行时崩溃:
```dart
// 推荐
point['name'] as String? ?? '途经点'

// 不推荐 (可能崩溃)
point['name'] as String
```

### 2. CI/CD 环境变量管理
敏感配置（API Key）使用 GitHub Secrets:
- 本地开发: `.env` 文件
- CI 构建: `secrets.AMAP_KEY`
- 避免硬编码密钥到代码库

### 3. 类型安全
Flutter 3.19+ 严格类型检查:
```dart
// 显式转换
elevation / maxElevation  // num
.toDouble()              // double
```
