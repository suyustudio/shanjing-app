## 6. 测试方法

### 6.1 性能测试工具

| 工具 | 用途 | 使用场景 |
|------|------|----------|
| Flutter DevTools | FPS/内存分析 | 开发阶段 |
| Android Studio Profiler | CPU/内存/网络 | 深度分析 |
| Xcode Instruments | iOS性能分析 | iOS专项 |
| Apache Bench | API压测 | 服务端性能 |
| Firebase Performance | 线上性能监控 | 生产环境 |

### 6.2 测试脚本

```dart
// FPS测试示例
Future<double> measureFPS({
  required Duration duration,
  required WidgetTester tester,
}) async {
  int frames = 0;
  final stopwatch = Stopwatch()..start();
  
  while (stopwatch.elapsed < duration) {
    await tester.pump(const Duration(milliseconds: 16));
    frames++;
  }
  
  return frames / duration.inSeconds;
}

// API响应时间测试示例
Future<int> measureApiResponse(String endpoint) async {
  final stopwatch = Stopwatch()..start();
  final response = await http.get(Uri.parse(endpoint));
  stopwatch.stop();
  return stopwatch.elapsedMilliseconds;
}
```

### 6.3 测试命令

```bash

flutter drive --profile --trace-startup test_driver/perf.dart

ab -n 10000 -c 100 https://api.shanjing.app/recommendations
```

|--------|----------|------|
| perf_launch_time | 启动完成 | duration_ms, is_cold |

## 9. 优化建议

### 9.1 新手引导优化

| 问题 | 优化方案 | 预期收益 |
