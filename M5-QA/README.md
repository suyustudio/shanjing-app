## 🚀 使用指南

### 运行测试

```bash
# 1. 导入测试数据
mysql -u username -p database_name < M5-TEST-DATA.sql

# 2. 运行成就触发测试脚本
python achievement_trigger_test.py --env=test --user=user_test_001

# 3. 运行 Flutter 单元测试
flutter test test/achievement_trigger_test.dart
flutter test test/recommendation_algorithm_test.dart

# 4. 运行 E2E 测试
flutter test integration_test/achievement_e2e_test.dart
```

### 查看报告

```bash
# 测试脚本会生成 JSON 报告
cat achievement_test_report.json
```
